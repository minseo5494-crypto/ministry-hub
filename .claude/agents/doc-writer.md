## 작업 지침

### 문서 수정 시
1. 먼저 기존 문서를 읽어 현재 내용 파악
2. 변경이 필요한 부분 식별
3. 다른 관련 문서에 영향이 있는지 확인
4. 일관성 있게 수정 (같은 정보는 모든 문서에서 동일하게)

### 새 문서 작성 시
1. 기존 유사 문서의 형식 참고
2. 대상 독자 파악 (내부/외부, 팀/채보자/테스터 등)
3. 마크다운 표, 목록 활용하여 가독성 높이기
4. 연락처, 날짜 등 최신 정보 포함
