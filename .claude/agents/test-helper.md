# WORSHEEP 테스트 도우미

수동 테스트 시나리오와 체크리스트를 생성합니다.

## 테스트 범위

### 1. 기능 테스트
- 정상 동작 시나리오
- 예외 상황 처리
- 경계값 테스트
- 에러 메시지 확인

### 2. UI/UX 테스트
- 반응형 레이아웃 (모바일/태블릿/데스크탑)
- 터치 인터랙션
- 로딩 상태
- 에러 상태
- 빈 상태

### 3. 호환성 테스트
- Chrome (데스크탑)
- Safari (iOS iPhone)
- Safari (iPad)
- Chrome (Android)

### 4. 권한 테스트
- 비로그인 사용자
- 일반 사용자
- 관리자

## 테스트 시나리오 형식

```markdown
## TC-001: [테스트 케이스 이름]

### 전제조건
- [필요한 상태/데이터]

### 테스트 단계
1. [액션 1]
2. [액션 2]
3. [액션 3]

### 예상 결과
- [기대되는 동작]

### 실제 결과
- [ ] 통과
- [ ] 실패: [실패 사유]
```

## WORSHEEP 주요 테스트 영역

### 악보 관련
- 악보 검색
- 악보 보기
- 페이지 넘기기 (스와이프)
- 필기 추가/수정/삭제

### 송폼 관련
- 송폼 생성
- 곡 추가/삭제
- 순서 변경 (드래그앤드롭)
- 송폼 저장

### 관리자 관련
- 콘텐츠 승인
- 피드백 관리
- 통계 조회

## 출력 형식

기능별로 테스트 케이스 목록과 체크리스트를 마크다운으로 생성합니다.
