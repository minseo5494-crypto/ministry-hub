
# Ⅳ. 시장 분석 및 성장 전략

WORSHEEP의 1차 타겟은 등록 교인 100명 이상 교회이다.

---

## 1. 시장 규모

### 1.1 국내 시장

| 구분 | 수치 |
|------|------|
| 한국 교회 수 | 약 60,000개 |
| 찬양팀 보유 교회 | 약 40,000개 (67%) |
| 팀당 평균 인원 | 8-15명 |
| 잠재 사용자 | **40만 ~ 60만명** |

**세분화:**
| 교회 규모 | 수 | 특징 |
|----------|-----|------|
| 메가처치 (1,000명+) | 500개 | 전문 사역팀, 높은 지불 의향 |
| 대형 (300-1,000명) | 3,000개 | 정규 찬양팀, 팀 관리 필요 |
| 중형 (100-300명) | 10,000개 | **핵심 타겟**, 효율성 추구 |
| 소형 (100명 미만) | 26,500개 | 무료 플랜 위주 |

### 1.2 글로벌 시장 (해외 한인교회)

| 지역 | 교회 수 | 특징 |
|------|---------|------|
| 미국 | 3,500개+ | 최대 시장, Planning Center 사용 경험 |
| 캐나다 | 400개+ | 미국과 유사 |
| 호주 | 300개+ | 빠른 디지털 도입 |
| 유럽 | 500개+ | 분산된 시장 |
| 기타 | 300개+ | 동남아, 중남미 등 |
| **합계** | **5,000개+** | 잠재 사용자 **5만명+** |

### 1.3 TAM/SAM/SOM

| 구분 | 정의 | 규모 |
|------|------|------|
| TAM | 전체 시장 | 65만명 (국내 60만 + 해외 5만) |
| SAM | 접근 가능 시장 | 20만명 (찬양팀 활성 멤버) |
| SOM | 초기 목표 시장 | 2만명 (중형 교회 중심) |

## 2. 경쟁 분석

### 2.1 직접 경쟁

| 서비스 | 특징 | 강점 | 약점 | 가격 |
|--------|------|------|------|------|
| **Planning Center** | 글로벌 1위 | 풍부한 기능, 안정성 | 영어 전용, 한국 CCM 없음, 비쌈 | $14-29/월 |
| **SongSelect (CCLI)** | 악보 라이센스 | 방대한 라이브러리 | 영어 위주, 협업 기능 없음 | $15/월 |
| **Praisecharts** | 악보 판매 | 고품질 악보 | 협업/콘티 없음 | 곡당 $3-15 |

### 2.2 간접 경쟁 (대체재)

| 서비스 | 용도 | 한계 |
|--------|------|------|
| **GoodNotes/Notability** | 악보 필기 | 팀 기능 없음, 악보 특화 아님 |
| **Google Drive** | 파일 공유 | 악보 관리 기능 없음 |
| **카카오톡** | 즉시 공유 | 파일 만료, 버전관리 불가 |
| **엑셀/워드** | 콘티 작성 | 비효율적, 공유 어려움 |

### 2.3 경쟁 우위

| 차별점 | 설명 |
|--------|------|
| **한국 시장 유일** | 한국 찬양팀 전용 올인원 서비스 없음 |
| **한국 CCM 특화** | 한글 UI, 한국 CCM 데이터 1,000곡+ |
| **올인원 통합** | 악보 + 콘티 + 필기 + 팀 관리 |
| **합리적 가격** | Planning Center 대비 **1/3 가격** |
| **AI 검색** | 자연어로 곡 검색 가능 |
| **저작권 정산** | Usage Score 기반 정당한 보상 |

## 3. 성장 전략

### 3.1 Phase 1: 시장 진입 (2026)

**타겟:** 중형 교회 찬양팀 (100-300명)

**전략:**
```
1. 무료 플랜으로 사용자 확보
2. 베타 테스터 피드백 반영
3. 교회 네트워크 활용 (연합회, 협의회)
4. 입소문 마케팅 (찬양팀 커뮤니티)
```

**채널:**
| 채널 | 방법 |
|------|------|
| 교회 협의회 | 연합 행사 홍보, 단체 구독 |
| 찬양팀 커뮤니티 | 온라인 커뮤니티, 밴드/카페 |
| 워십 컨퍼런스 | 부스 운영, 데모 시연 |
| 워십팀 직접 컨택 | 파트너십 제안 |

### 3.2 Phase 2: 시장 확대 (2027)

**타겟:** 대형 교회 + 메가처치

**전략:**
```
1. 성공 사례 기반 마케팅
2. 엔터프라이즈 기능 추가
3. 공식 파트너십 확대 (워십팀)
4. 교단/연합회 단체 계약
```

**추가 기능:**
- 다중 팀 관리 (교회 내 여러 찬양팀)
- 고급 분석 대시보드
- API 연동 (교회 관리 시스템)

### 3.3 Phase 3: 글로벌 진출 (2027-2028)

**타겟:** 해외 한인교회

**전략:**
```
1. 영어 UI 지원
2. 해외 한인교회 네트워크 활용
3. 글로벌 CCM 데이터 확보
4. 현지 워십팀 파트너십
```

**우선순위:**
1. 미국 (LA, NY, 시카고)
2. 캐나다 (토론토, 밴쿠버)
3. 호주 (시드니, 멜버른)

### 3.4 Phase 4: 플랫폼 진화 (2028+)

**목표:** 찬양 사역 데이터 플랫폼

**확장 서비스:**
| 서비스 | 설명 |
|--------|------|
| 추천 시스템 | 교회/팀별 맞춤 곡 추천 |
| 큐레이션 | 시즌/테마별 곡 모음 |
| 사역 분석 | 팀 활동 리포트 |
| 악보 마켓플레이스 | 편곡 악보 판매/구매 |
| 스트리밍 | 자체 악보 스트리밍 |

## 4. 마케팅 전략

### 4.1 콘텐츠 마케팅

| 콘텐츠 | 채널 | 목적 |
|--------|------|------|
| 찬양팀 운영 팁 | 블로그, 유튜브 | SEO, 브랜드 인지도 |
| 예배 준비 가이드 | 이메일 뉴스레터 | 리드 육성 |
| 사용 사례 | 인스타그램, 유튜브 | 사회적 증거 |
| 워십 트렌드 | 블로그 | 업계 리더십 |

### 4.2 커뮤니티 마케팅

**전략:**
- 찬양팀 리더 네트워크 구축
- 온라인 커뮤니티 운영 (카카오톡 채널)
- 오프라인 모임 후원 (워십나잇 등)
- 레퍼럴 프로그램 (추천 보상)

### 4.3 B2B 마케팅

**대상:** 교단, 연합회, 대형 교회

**방법:**
- 맞춤 프레젠테이션
- 파일럿 프로그램 제공
- 단체 할인 (10팀 이상 30% 할인)
- 온보딩 지원

## 5. 진입 장벽 구축

### 5.1 데이터 장벽

```
사용자 축적 → 필기/콘티 데이터 축적 → 이탈 비용 증가
```

- 개인 필기 데이터 (다른 플랫폼으로 이전 어려움)
- 팀 콘티 히스토리
- 팀 악보 라이브러리

### 5.2 네트워크 효과

```
사용자 증가 → 팀 협업 가치 증가 → 더 많은 팀 가입
```

- 같은 교회 내 팀원들 간 협업
- 다른 교회 찬양팀과 악보 공유
- 워십팀 공식 악보 제공

### 5.3 콘텐츠 장벽

```
파트너십 확대 → 독점 악보 확보 → 차별화
```

- 워십팀 공식 악보 파트너십
- UGC 악보 축적
- 한국 CCM 특화 데이터

## 7. 리스크 및 대응

| 리스크 | 가능성 | 영향 | 대응 |
|--------|--------|------|------|
| 경쟁자 진입 | 중 | 높음 | 빠른 시장 선점, 데이터 장벽 구축 |
| 사용자 확보 실패 | 중 | 높음 | 무료 기능 강화, 교회 네트워크 활용 |
| Planning Center 한국 진출 | 낮음 | 높음 | 한국 CCM 특화, 가격 경쟁력 |
| 저작권 분쟁 | 중 | 중 | 사전 파트너십, Safe Harbor 정책 |

WORSHEEP는 초기에는 중형 교회를 중심으로 도입을 확산시키고,
이후 대형 교회, 메가처치로 확장하며,
국내 시장 안정화 이후에는 해외 한인교회를 중심으로 글로벌 확장을 추진한다.

장기적으로는 찬양 사역 데이터를 기반으로 한 플랫폼으로 진화하는 것을 목표로 한다.
