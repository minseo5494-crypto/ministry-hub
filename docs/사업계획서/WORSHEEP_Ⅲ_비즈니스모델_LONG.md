
# Ⅲ. 비즈니스 모델 및 수익 구조

WORSHEEP의 비즈니스 모델은 개인/팀/교회 3단계 구독형 SaaS와
콘텐츠 마켓 기반 수익 분배 구조를 결합한 형태이다.

---

## 1. 수익 구조 개요

WORSHEEP는 세 가지 수익원을 운영한다.

```
┌────────────────────────────────────────────────────────────┐
│                   WORSHEEP 수익 구조                        │
├──────────────────┬──────────────────┬──────────────────────┤
│  구독료 (1차)     │  콘텐츠 마켓 (2차) │  기관/단체 계약      │
├──────────────────┼──────────────────┼──────────────────────┤
│  • 개인 (Free/Pro)│  • 개별 악보 패키지 │  • 교단 단체 계약    │
│  • 팀 (Basic/Pro) │  • 시즌 패키지     │  • 선교단체 협의     │
│  • 교회 (S/M/L)   │  • 큐레이션 콘텐츠  │  • 연합회 제휴       │
└──────────────────┴──────────────────┴──────────────────────┘
```

## 2. 구독 요금제

### 2.1 개인 요금제

| 플랜 | 가격 | 주요 기능 |
|------|------|----------|
| **Free** | 무료 | 검색, 미리보기, 월 10곡 다운로드 |
| **Pro** | **월 ₩4,900** | 무제한 다운로드, 콘티 제작, 악보 필기 |

**Free → Pro 전환 유도:**
- 월 10곡 다운로드 제한
- 콘티/필기 기능 필요 시

### 2.2 팀 요금제

| 플랜 | 가격 | 대상 |
|------|------|------|
| **Team Basic** | **연 ₩149,000** | 소형팀 (5명 이하) |
| **Team Pro** | **연 ₩299,000** | 중형팀 (10명 이하) |

**팀 요금제 특징:**
- 팀원 전체 Pro 기능 사용
- 팀 악보 라이브러리 공유
- 콘티 협업 기능

### 2.3 교회 요금제

| 플랜 | 가격 | 팀 수 |
|------|------|-------|
| **Church S** | **연 ₩590,000** | ~10팀 |
| **Church M** | **연 ₩990,000** | ~25팀 |
| **Church L** | **연 ₩1,490,000** | 무제한 |

**교회 요금제 특징:**
- 교회 내 모든 찬양팀 통합 관리
- 멀티 관리자 기능
- 교회 전용 대시보드
- 우선 고객지원

## 4. 로열티 수익 구조

### 4.1 순수익 계산

```
순수익 = 총매출 - (결제수수료 + 서버비 + CDN + 스토리지 + 고객지원 인건비)
```

### 4.2 기본 배분

| 대상 | 배분율 |
|------|--------|
| 콘텐츠팀 (워십팀/채보자) | **45%** |
| 플랫폼 | **55%** |

### 4.3 프리미엄 파트너 등급

| 등급 | 콘텐츠팀 배분율 | 조건 |
|------|----------------|------|
| **Standard** | 45% | 기본 |
| **Premium** | 55% | 독점 제공, 홍보 참여 |
| **Incubation** | 60% (1년) | 신규 팀 육성 프로그램 |

### 4.4 정산 기준

- **정산 주기**: 월별 집계, 분기별 정산
- **기준**: Usage Score (열람, 다운로드, 콘티 등록, 예배 사용)
- **최소 정산액**: ₩10,000 이상 시 지급

## 8. 윈윈 구조

WORSHEEP의 비즈니스 모델은 모든 참여자가 이익을 얻는 구조이다.

| 참여자 | 기존 상황 | WORSHEEP |
|--------|----------|----------|
| **개인** | 악보 찾기 어려움 | 1,700곡+ 검색, AI 추천 |
| **찬양팀** | 비효율적 관리 | 팀 협업, 콘티 공유 |
| **교회** | 저작권 침해 우려 | 합법적 사용, 통합 관리 |
| **콘텐츠팀** | 악보 수익 없음 | Usage Score 기반 45~60% 수익 |
| **채보자** | 무료 봉사 | 수익 분배 참여 |

이를 통해 WORSHEEP는 개인, 팀, 교회, 콘텐츠팀 모두가 윈윈하는 구조를 만든다.
