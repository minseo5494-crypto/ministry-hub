
# Ⅲ. 비즈니스 모델 및 수익 구조

WORSHEEP의 비즈니스 모델은 개인/팀/교회 3단계 구독형 SaaS와
콘텐츠 마켓 기반 수익 분배 구조를 결합한 형태이다.

---

## 1. 수익 구조 개요

WORSHEEP는 세 가지 수익원을 운영한다.

```
┌────────────────────────────────────────────────────────────┐
│                   WORSHEEP 수익 구조                        │
├──────────────────┬──────────────────┬──────────────────────┤
│  구독료 (1차)     │  콘텐츠 마켓 (2차) │  기관/단체 계약      │
├──────────────────┼──────────────────┼──────────────────────┤
│  • 개인 (Free/Pro)│  • 개별 악보 패키지 │  • 교단 단체 계약    │
│  • 팀 (Basic/Pro) │  • 시즌 패키지     │  • 선교단체 협의     │
│  • 교회 (S/M/L)   │  • 큐레이션 콘텐츠  │  • 연합회 제휴       │
└──────────────────┴──────────────────┴──────────────────────┘
```

## 4. 로열티 수익 구조

### 4.1 사역 수익 풀 (Royalty Pool)

WORSHEEP는 구독 및 광고 수익을 포함한 플랫폼 전체 수익 중 일부를
악보 콘텐츠 정산을 위한 **사역 수익 풀(Royalty Pool)**로 운영한다.

```
┌─────────────────────────────────────────────────────────────────┐
│  사역 수익 풀 = 플랫폼 전체 수익의 일부                          │
│                                                                 │
│  ※ 특정 수익 항목(구독료, 광고 등)을 직접 분리하지 않음          │
│  ※ 악보 사용 행위 데이터 기반 정산 지수에 따라 배분              │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 정산 지수 (Settlement Index)

정산 지수는 단순 조회 수가 아닌,
실제 예배 준비 및 사용과 밀접한 행위를 중심으로 산정된다.

| 행동 | 설명 |
|------|------|
| 악보 열람 | 악보 상세 페이지 조회 |
| 악보 다운로드 | PDF/이미지 다운로드 |
| 악보 필기 및 주석 | 필기 도구 사용, 메모 추가 |
| 콘티 포함 | 해당 악보가 콘티에 등록됨 |
| 예배 실행 | 예배일 기준 콘티 실행 (스트리밍) |

### 4.3 좋아요(선호도) 지표

'좋아요'는 악보에 대한 이용자의 선호와 인기도를 나타내는 **보조 지표**로 활용된다.

- 좋아요는 단독으로 정산 금액을 결정하지 않음
- 정산 지수 산정 시 참고 요소 또는 보정 계수로만 반영
- 단순 클릭이나 어뷰징에 의해 수익이 왜곡되지 않도록 설계

### 4.4 조건부 수익 비율 상향

사역팀은 기본 정산 비율 외에도, 다음 조건 충족 시
사역 수익 풀 내에서 **최대 70%**까지 수익 비율 상향이 가능하다.

| 조건 | 설명 |
|------|------|
| 신곡/편곡 우선 공개 | WORSHEEP에서 최초 또는 우선 공개 |
| 공식 악보 제공 | 팀 직접 제공 공식 악보 |
| 홍보 협력 | 플랫폼 마케팅/홍보 활동 참여 |

이는 플랫폼과 사역팀 간의 **선택적·상호 이익 구조**로 설계되었으며,
강제 조건 없이 자율적으로 참여할 수 있다.

### 4.5 정산 기준

- **정산 주기**: 월별 집계, 분기별 정산
- **기준**: 정산 지수 (Settlement Index)
- **최소 정산액**: ₩10,000 이상 시 지급

**유의사항:**
- 정산 지수는 상대 평가 방식으로 산정
- 플랫폼 정책, 서비스 구조, 이용 패턴 변화에 따라 조정 가능
- 조정 사항은 사전 고지되며, 기발생 정산에 소급 적용되지 않음

## 8. 윈윈 구조

WORSHEEP의 비즈니스 모델은 모든 참여자가 이익을 얻는 구조이다.

| 참여자 | 기존 상황 | WORSHEEP |
|--------|----------|----------|
| **개인** | 악보 찾기 어려움 | 1,700곡+ 검색, AI 추천 |
| **찬양팀** | 비효율적 관리 | 팀 협업, 콘티 공유 |
| **교회** | 저작권 침해 우려 | 합법적 사용, 통합 관리 |
| **콘텐츠팀** | 악보 수익 없음 | 정산 지수 기반 최대 70% 수익 |
| **채보자** | 무료 봉사 | 수익 분배 참여 |

이를 통해 WORSHEEP는 개인, 팀, 교회, 콘텐츠팀 모두가 윈윈하는 구조를 만든다.
