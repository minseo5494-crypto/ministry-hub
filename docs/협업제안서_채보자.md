# WORSHEEP 채보자 협업 제안서

**찬양팀을 위한 통합 악보 관리 플랫폼**

2026년 1월

---

## 인사말

안녕하세요,

먼저 오랜 시간 동안 찬양 악보를 채보하시고 공유해 주신 노력에 깊은 감사의 말씀을 드립니다.

블로그에 올려주신 악보들 덕분에 수많은 교회 찬양팀들이 예배를 준비하는 데 큰 도움을 받아왔습니다. 저 역시 찬양팀 사역을 하면서 올려주신 악보들을 많이 사용했기에, 그 수고로움을 조금이나마 알고 있습니다.

저희는 한국 교회 찬양팀들의 악보 관리와 예배 준비를 돕기 위한 **WORSHEEP**이라는 플랫폼을 운영하고 있습니다. 채보자님의 악보를 더 많은 찬양팀들에게 제공하고, 그 노력에 대한 보상이 이루어질 수 있도록 협력을 제안드리고자 합니다.

## 1. 핵심 요약

### 요청드리는 것

> **채보하신 악보를 WORSHEEP에서 제공할 수 있도록 허락해 주세요.**

### 대상이 되는 악보

저희가 요청드리는 악보는 다음 조건에 해당하는 곡입니다:

| 조건 | 설명 |
|------|------|
| **저작권 허락 완료** | 원곡 팀(저작권자)에게 사용 허락을 받은 곡 |
| **퍼블릭 도메인** | 저작권이 만료된 곡 (저작자 사후 70년 경과) |

> **중요**: 저희는 원곡 팀에게 먼저 저작권 사용 허락을 받고,
> 그 다음에 해당 곡의 채보 악보 사용을 요청드립니다.

### 채보자님이 얻는 것

| 혜택 | 설명 |
|------|------|
| **수익 분배** | 구독 수익의 일정 비율을 채보자에게 정산 |
| **채보자 표기** | 모든 악보에 채보자 정보 표시 |
| **노출 확대** | 전국 교회 찬양팀에게 악보 제공 |
| **무료 프리미엄** | 협력 채보자에게 프리미엄 기능 무료 제공 |
| **채보자 프로필** | 채보자 소개 페이지 제공 (선택) |

### 채보자님이 해주실 일

> **사용 동의와 악보 파일 제공뿐입니다.**

## 2. 왜 이 제안을 드리는가

### 2.1 채보 악보의 가치

솔직히 말씀드리면, 많은 CCM 곡들은 **공식 악보가 없거나 구하기 어렵습니다.**

특히 2010년대 이전 곡들, 소규모 팀의 곡들은:
- 팀에서 공식 악보를 배포하지 않는 경우가 많습니다
- 설령 있어도 찾기가 매우 어렵습니다
- 결국 **채보 악보가 사실상 유일한 악보**인 경우가 많습니다

채보자님이 만드신 악보는 단순한 '비공식 악보'가 아니라,
**실제로 교회 현장에서 사용되는 유일한 악보**입니다.

### 2.2 현재의 문제점

하지만 현재 상황은 채보자님의 노력에 비해 보상이 적습니다:

| 문제 | 현실 |
|------|------|
| **무단 복사** | 블로그 악보가 출처 없이 재배포됩니다 |
| **수익 없음** | 수많은 사람들이 사용하지만 보상이 없습니다 |
| **통계 없음** | 얼마나 많은 사람들이 사용하는지 모릅니다 |
| **관리 부담** | 여러 블로그/사이트에 분산 관리해야 합니다 |

### 2.3 WORSHEEP이 드리는 것

WORSHEEP과 협력하시면:

```
[현재 상황]
채보 → 블로그 업로드 → 무단 복사 → 출처 사라짐 → 보상 없음

[WORSHEEP 협력]
채보 → WORSHEEP 등록 → 채보자 표기 → 사용량 추적 → 수익 분배
```

## 3. 협력의 구체적인 내용

### 3.1 저희가 하는 것

| 항목 | 내용 |
|------|------|
| **악보 등록** | 제공해주신 악보를 플랫폼에 등록 |
| **채보자 표기** | 모든 악보에 "채보: [채보자명]" 표시 |
| **저작권 처리** | 원곡 팀에게 저작권 사용 허락 확보 (저희가 진행) |
| **수익 정산** | 사용량에 따른 수익 분배 |
| **통계 제공** | 조회수, 다운로드 수 등 사용 데이터 제공 |

### 3.2 저희가 하지 않는 것

| 항목 | 설명 |
|------|------|
| **채보 저작권 침해** | 채보자님의 편곡/채보 권리를 존중합니다 |
| **무단 수정** | 악보를 임의로 수정하지 않습니다 |
| **독점 요구** | 기존 블로그 게시를 삭제하실 필요 없습니다 |

### 3.3 수익 구조

WORSHEEP의 수익 분배 구조:

```
유료 구독 수익
    ├─ 50% → 저작권자 (원곡 팀)
    ├─ 20% → 채보자
    └─ 30% → 플랫폼 운영
```

> **참고**: 퍼블릭 도메인 곡의 경우 저작권자 몫이 없으므로,
> 채보자 정산 비율이 높아질 수 있습니다. (협의 가능)

### 3.4 대상 악보 범위

저희가 요청드리는 악보는 **저작권 처리가 가능한 곡**입니다:

| 상황 | 진행 방식 |
|------|----------|
| **원곡 팀과 파트너십 있음** | 바로 채보 악보 등록 가능 |
| **원곡 팀과 파트너십 없음** | 저희가 먼저 팀에 연락하여 허락 확보 후 진행 |
| **퍼블릭 도메인 곡** | 바로 채보 악보 등록 가능 |
| **저작권 허락 불가한 곡** | 해당 곡은 등록 불가 |

> 채보자님이 직접 저작권 팀에 연락하실 필요 없습니다.
> 저희가 먼저 원곡 팀과 협의를 진행합니다.

### 3.5 원곡 팀의 사전 동의

저희는 원곡 팀과 파트너십을 맺을 때, **채보 악보에 대한 동의도 함께 받습니다.**

원곡 팀이 동의하는 내용:
> "공식 악보가 없는 곡에 대해 채보자가 만든 악보를 WORSHEEP에서 제공하는 것에 동의합니다."

따라서 파트너십이 체결된 팀의 곡은:
- 원곡 팀이 **이미 채보 악보 등록에 동의**한 상태입니다
- 채보자님의 동의만 있으면 **바로 등록 가능**합니다
- 수익은 **사전에 합의된 비율**로 자동 분배됩니다

이렇게 하는 이유:
- 채보자님이 일일이 원곡 팀에 연락할 필요가 없습니다
- 저작권 문제로 인한 분쟁을 사전에 방지합니다
- 모든 당사자가 공정하게 수익을 나눕니다

## 4. 자주 묻는 질문

### "제가 채보한 건데, 왜 원곡 팀 허락이 필요한가요?"

채보 악보에는 두 가지 권리가 있습니다:

| 권리 | 소유자 | 설명 |
|------|--------|------|
| **원곡 저작권** | 작곡가/팀 | 멜로디, 코드 진행, 가사에 대한 권리 |
| **채보/편곡권** | 채보자님 | 악보로 정리한 노력과 편곡에 대한 권리 |

채보 악보를 공식적으로 배포하려면 **두 권리 모두** 허락이 필요합니다.
그래서 저희는:
1. 먼저 원곡 팀에게 저작권 사용 허락을 받고
2. 그 다음 채보자님께 채보 악보 사용 허락을 요청드립니다

### "블로그에 이미 올려놓은 악보도 되나요?"

**네, 됩니다.**

기존에 블로그에 올리신 악보도 사용 가능합니다.
블로그 게시물을 삭제하실 필요도 없습니다.

### "제가 해야 할 일이 뭔가요?"

1. **사용 동의**: 이메일 또는 간단한 동의서
2. **악보 파일 제공**: 원본 파일 (PDF, 악보 프로그램 파일 등)
3. **정보 확인**: 곡 제목, 원곡 팀 확인

등록, 관리, 저작권 협의는 **저희가 담당**합니다.

### "수익이 실제로 발생할까요?"

솔직히 말씀드리면, 초기에는 수익이 크지 않을 수 있습니다.

하지만:
- 현재는 **0원**입니다
- WORSHEEP과 협력하시면 **사용량에 따른 정산**이 발생합니다
- 서비스가 성장할수록 정산 금액도 늘어납니다

또한 수익 외에도:
- 채보자로서 **정당한 인정**을 받고
- **정확한 사용 통계**를 확인할 수 있습니다

### "독점 계약인가요? 블로그 글 삭제해야 하나요?"

**아닙니다.**

- 기존 블로그 게시물을 유지하셔도 됩니다
- 다른 곳에 악보를 올리셔도 됩니다
- WORSHEEP은 **추가 채널**입니다

## 5. 협력 절차

### Step 1: 곡 목록 확인

채보자님이 채보하신 곡 목록을 확인합니다.

### Step 2: 저작권 확보 (저희가 진행)

해당 곡들의 원곡 팀에 저작권 사용 허락을 요청합니다.
- 이미 파트너십이 있는 팀: 바로 진행
- 파트너십이 없는 팀: 저희가 연락하여 협의
- 퍼블릭 도메인: 바로 진행 가능

### Step 3: 채보 악보 사용 동의

저작권이 확보된 곡에 대해 채보 악보 사용 동의를 요청드립니다.

### Step 4: 악보 등록

동의하신 악보를 플랫폼에 등록합니다.
- 채보자 정보 표기
- 원곡 팀 정보 표기

### Step 5: 정산

정식 서비스 출시 후 사용량에 따라 정산합니다.

## 6. 정리

### 요약

1. **채보자님의 악보는 가치가 있습니다** - 실제로 교회에서 사용되는 유일한 악보인 경우가 많습니다
2. **저작권은 저희가 처리합니다** - 원곡 팀 허락은 저희가 받아옵니다
3. **채보자로서 인정받습니다** - 모든 악보에 채보자 정보가 표시됩니다
4. **수익을 분배받습니다** - 사용량에 따른 정산이 이루어집니다
5. **기존 활동을 유지할 수 있습니다** - 블로그 삭제 불필요, 독점 아님

### 요청드리는 것

> **채보하신 악보를 WORSHEEP에서 제공할 수 있도록 허락해 주세요.**

오랜 시간 찬양 사역을 위해 악보를 만들어 오신 노력이
정당하게 인정받고, 보상받을 수 있는 환경을 함께 만들고 싶습니다.

관심을 가져주셔서 감사합니다.
추가 문의사항이나 협력 의향이 있으시면 아래 연락처로 연락 주시기 바랍니다.
