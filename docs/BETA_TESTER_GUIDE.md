## 환영합니다!

WORSHEEP 베타 테스트에 참여해 주셔서 감사합니다.

WORSHEEP은 **찬양팀을 위한 올인원 악보 관리 플랫폼**입니다.
여러분의 소중한 피드백으로 더 좋은 서비스를 만들어 나가겠습니다.

## 1. 시작하기

### 1.1 접속 주소

```
https://worsheep.vercel.app
```

> 아이패드, 모바일, PC 모두 사용 가능합니다.
> **아이패드**에서 사용하시면 필기 기능을 최대한 활용할 수 있습니다.

### 1.2 회원가입

1. 우측 상단 **[로그인]** 클릭
2. **[회원가입]** 클릭
3. 가입 방법 선택:
   - **이메일 가입**: 이메일, 비밀번호, 이름 입력
   - **Google 가입**: Google 계정으로 빠른 가입
4. 이용약관 및 개인정보처리방침 동의
5. **[가입하기]** 클릭

### 1.3 로그인

1. 우측 상단 **[로그인]** 클릭
2. 이메일/비밀번호 입력 또는 Google 로그인
3. 로그인 완료!

### 1.4 비밀번호 찾기

1. 로그인 화면에서 **[비밀번호를 잊으셨나요?]** 클릭
2. 가입 시 사용한 이메일 입력
3. 이메일로 받은 링크에서 비밀번호 재설정

## 2. 악보 검색하기

### 2.1 기본 검색

메인 화면 상단의 검색창에서 검색할 수 있습니다.

**검색 가능 항목:**
- 곡 제목 (예: "감사와 찬양")
- 아티스트/팀명 (예: "마커스워십")
- 가사 일부 (예: "주님의 사랑이")

### 2.2 필터 검색

검색창 아래의 필터를 사용하면 더 정확하게 찾을 수 있습니다.

| 필터 | 설명 | 예시 |
|------|------|------|
| **절기** | 교회력 절기 | 성탄절, 부활절, 추수감사절 |
| **테마** | 곡의 주제 | 찬양, 경배, 회개, 위로 |
| **조성** | 악보 키 | C, D, E, G, A 등 |
| **박자** | 박자표 | 4/4, 6/8, 3/4 |
| **템포** | 빠르기 | 느림, 보통, 빠름 |
| **BPM** | 정확한 빠르기 | 60~200 |

### 2.3 AI 검색 (추천)

검색창 옆의 **[AI 검색]** 토글을 켜면 자연어로 검색할 수 있습니다.

**예시:**
- "부활절에 부르기 좋은 곡"
- "빠른 템포의 찬양곡"
- "G키 경배곡 추천해줘"
- "어노인팅 곡 중에서 느린 곡"

## 3. 악보 보기 & 다운로드

### 3.1 악보 미리보기

1. 검색 결과에서 원하는 곡 클릭
2. 곡 상세 페이지에서 **[악보 보기]** 클릭
3. PDF 악보가 화면에 표시됩니다

**악보 뷰어 조작법:**
- **페이지 넘기기**: 좌우 화살표 버튼 또는 스와이프
- **확대/축소**: 핀치 줌 (터치) 또는 +/- 버튼
- **전체화면**: 전체화면 버튼 클릭

### 3.2 악보 다운로드

악보 뷰어에서 다운로드 버튼을 클릭하면:

| 형식 | 설명 | 용도 |
|------|------|------|
| **PDF** | 원본 악보 파일 | 인쇄, 보관 |
| **이미지** | PNG 이미지 파일 | 카톡 공유, SNS |
| **PPT** | 파워포인트 슬라이드 | 프로젝터 송출 |

### 3.3 좋아요

곡 상세 페이지에서 **하트(♥) 버튼**을 누르면 좋아요가 저장됩니다.
좋아요한 곡은 **마이페이지 > 좋아요한 곡**에서 확인할 수 있습니다.

## 4. 악보에 필기하기

### 4.1 필기 모드 진입

1. 악보 뷰어에서 **[필기 모드]** 또는 **연필 아이콘** 클릭
2. 악보 에디터가 열립니다

### 4.2 필기 도구

| 도구 | 설명 | 사용법 |
|------|------|--------|
| **펜** | 기본 필기 | 악보 위에 자유롭게 그리기 |
| **형광펜** | 하이라이트 | 중요 부분 강조 |
| **지우개** | 지우기 | 그린 내용 삭제 |
| **텍스트** | 글자 입력 | 코드, 메모 추가 |
| **실행취소** | 되돌리기 | 방금 그린 것 취소 |

### 4.3 색상 & 굵기

- 도구 선택 후 **색상 팔레트**에서 원하는 색 선택
- **굵기 슬라이더**로 펜 굵기 조절

### 4.4 송폼 설정

악보에 **송폼 태그**를 추가할 수 있습니다.

**사용 가능한 송폼:**
```
Intro | Verse | Pre-Chorus | Chorus | Bridge | Outro | Interlude | Tag
```

1. **[송폼]** 버튼 클릭
2. 원하는 송폼 선택
3. 악보 위에 드래그하여 배치
4. 크기, 색상 조절 가능

### 4.5 드럼/피아노 악보

기본 악보 외에 **드럼 악보**와 **피아노 악보**를 추가할 수 있습니다.

1. 에디터 상단의 **탭**에서 [드럼] 또는 [피아노] 선택
2. 마디 추가, 음표 입력
3. 드래그로 마디 너비 조절

### 4.6 필기 저장

- 필기 내용은 **자동 저장**됩니다
- 명시적으로 저장하려면 **[저장]** 버튼 클릭
- 저장된 필기는 다른 기기에서도 동기화됩니다

### 4.7 필기 내보내기

에디터에서 **[내보내기]** 버튼을 클릭하면:
- **PDF**: 필기 포함된 악보 PDF
- **이미지**: 필기 포함된 PNG 이미지

## 5. 팀 만들기 & 참여하기

### 5.1 새 팀 만들기

1. 상단 메뉴에서 **[내 팀]** 클릭
2. **[새 팀 만들기]** 클릭
3. 팀 정보 입력:
   - 팀 이름 (예: "○○교회 찬양팀")
   - 팀 설명 (선택)
4. **[만들기]** 클릭
5. **초대 코드**가 생성됩니다 - 팀원들에게 공유하세요!

### 5.2 팀 참여하기

1. 상단 메뉴에서 **[내 팀]** 클릭
2. **[팀 참여하기]** 클릭
3. 리더에게 받은 **초대 코드 6자리** 입력
4. **[참여하기]** 클릭

### 5.3 팀 관리 (리더용)

팀 페이지에서 **[설정]** 클릭:

| 기능 | 설명 |
|------|------|
| **멤버 관리** | 멤버 목록 확인, 직책 변경, 내보내기 |
| **직책 설정** | 커스텀 직책 추가 (인도자, 드럼, 베이스 등) |
| **초대 코드** | 새 초대 코드 생성 |
| **고정곡 관리** | 팀에서 자주 쓰는 곡 등록 |
| **팀 정보 수정** | 팀 이름, 설명 변경 |
| **팀 삭제** | 팀 완전 삭제 (주의!) |

### 5.4 고정곡

팀에서 자주 사용하는 곡을 **고정곡**으로 등록하면 빠르게 접근할 수 있습니다.

1. 팀 페이지에서 **[고정곡]** 탭 선택
2. **[곡 추가]** 클릭
3. 검색하여 곡 선택
4. 고정곡 목록에 추가됨

## 6. 콘티 만들기

### 6.1 새 콘티 만들기

1. 팀 페이지에서 **[콘티]** 탭 선택
2. **[새 콘티 만들기]** 클릭
3. 콘티 정보 입력:
   - 제목 (예: "2026년 2월 2일 주일예배")
   - 날짜
4. **[만들기]** 클릭

### 6.2 곡 추가하기

콘티 상세 페이지에서:

1. **[곡 추가]** 클릭
2. 검색하여 곡 선택
3. 곡이 콘티에 추가됨

**여러 곡 한번에 추가:**
- 검색 결과에서 여러 곡 체크
- **[선택한 곡 추가]** 클릭

### 6.3 순서 변경하기

**드래그 앤 드롭**으로 곡 순서를 변경할 수 있습니다.

1. 곡 왼쪽의 **핸들(≡)** 을 길게 누르기
2. 원하는 위치로 드래그
3. 놓으면 순서 변경 완료

### 6.4 곡별 설정

각 곡 옆의 **[...]** 메뉴에서:

| 옵션 | 설명 |
|------|------|
| **키 변경** | 원키 대신 다른 키로 표시 |
| **메모 추가** | 곡별 메모 (예: "2절만", "느리게") |
| **악보 보기** | 해당 곡 악보로 이동 |
| **삭제** | 콘티에서 곡 제거 |

### 6.5 콘티 공유

팀 콘티는 **팀원 모두가 볼 수 있습니다**.
별도의 공유 설정 없이 팀에 가입된 모든 멤버가 접근 가능합니다.

### 6.6 콘티에서 악보 보기

1. 콘티 상세 페이지에서 **[악보 보기]** 클릭
2. 콘티 순서대로 악보가 연결되어 표시됩니다
3. 스와이프로 다음 곡 악보로 이동

## 7. 개인 콘티 & 필기 관리

### 7.1 개인 콘티

팀 콘티와 별개로 **개인만의 콘티**를 만들 수 있습니다.

1. 상단 메뉴에서 **[개인 콘티]** 클릭
2. **[새 콘티]** 만들기
3. 팀 콘티와 동일하게 곡 추가, 순서 변경 가능

### 7.2 내 필기 노트

저장한 필기 노트를 한눈에 관리할 수 있습니다.

1. 상단 메뉴에서 **[내 필기]** 클릭
2. 저장된 필기 노트 목록 확인
3. 클릭하면 해당 필기 에디터로 이동

**필기 삭제:**
- 각 노트 옆의 **[삭제]** 버튼 클릭
- 삭제된 노트는 복구할 수 없으니 주의!

### 7.3 마이페이지

상단 프로필 아이콘 > **[마이페이지]**에서:

| 탭 | 내용 |
|----|------|
| **업로드한 곡** | 내가 업로드한 곡 목록 |
| **좋아요한 곡** | 하트 누른 곡 목록 |
| **내 필기** | 저장된 필기 노트 |
| **설정** | 프로필 수정, 로그아웃 |

## 8. 곡 업로드하기

### 8.1 새 곡 업로드

1. 상단 메뉴에서 **[업로드]** 클릭
2. 곡 정보 입력:

| 항목 | 필수 | 설명 |
|------|------|------|
| 곡 제목 | O | 정확한 곡명 |
| 아티스트 | O | 원곡 아티스트/팀 |
| 키(조성) | O | 악보의 키 |
| BPM | X | 곡의 빠르기 |
| 박자 | X | 4/4, 6/8 등 |
| 테마 | X | 찬양, 경배 등 |
| 절기 | X | 성탄절, 부활절 등 |
| YouTube | X | 유튜브 영상 링크 |

3. **악보 파일 업로드** (PDF 또는 이미지)
4. **[업로드]** 클릭

### 8.2 대량 업로드

여러 곡을 한번에 업로드할 수 있습니다.

1. **[대량 업로드]** 탭 선택
2. **Excel 템플릿 다운로드**
3. 템플릿에 곡 정보 입력
4. Excel 파일 업로드
5. 악보 파일들 업로드 (파일명이 곡 제목과 일치해야 함)

### 8.3 업로드 후 승인

> 업로드된 곡은 **관리자 승인 후** 검색에 노출됩니다.
> 승인까지 1~2일 정도 소요될 수 있습니다.

## 9. 피드백 보내기

여러분의 피드백이 WORSHEEP을 더 좋게 만듭니다!

### 9.1 인앱 피드백 버튼

화면 **우측 하단**의 **파란색 버튼**을 클릭하세요.

1. 피드백 유형 선택:
   - **버그 신고** 🐛: 오류, 에러 발견 시
   - **기능 제안** 💡: 새로운 기능 아이디어
   - **기타 의견** ❓: 그 외 모든 의견
2. 내용 작성
3. **[보내기]** 클릭

> 현재 페이지 URL과 기기 정보가 자동으로 포함되어
> 문제 해결에 큰 도움이 됩니다!

### 9.2 피드백 작성 팁

**좋은 버그 리포트 예시:**
```
[상황] 콘티에서 곡 순서를 변경하려고 했습니다.
[문제] 드래그 후 놓으면 원래 위치로 돌아갑니다.
[환경] 아이패드 Safari, iOS 17
```

**좋은 기능 제안 예시:**
```
[제안] 콘티를 PDF로 내보내기 기능이 있으면 좋겠습니다.
[이유] 예배 전에 팀원들에게 카톡으로 공유하기 위해서입니다.
```

### 9.3 카카오톡 오픈채팅

실시간 소통을 위한 오픈채팅방:


## 10. 자주 묻는 질문

### Q1. 아이패드에서 Apple Pencil로 필기가 안 돼요

**A:** 필기 모드로 진입했는지 확인해주세요.
1. 악보 뷰어에서 **연필 아이콘** 클릭
2. 필기 도구(펜, 형광펜 등) 선택
3. 이제 Apple Pencil로 필기 가능!

### Q2. 저장한 필기가 사라졌어요

**A:** 필기는 자동 저장되지만, 다음을 확인해주세요:
1. 로그인된 상태인지 확인
2. 같은 계정으로 로그인했는지 확인
3. 인터넷 연결 상태 확인

문제가 지속되면 피드백 버튼으로 알려주세요!

### Q3. 팀 초대 코드가 작동하지 않아요

**A:**
1. 초대 코드가 정확히 6자리인지 확인
2. 대소문자 구분 없음
3. 코드가 만료되었을 수 있음 - 리더에게 새 코드 요청

### Q4. 악보 다운로드가 안 돼요

**A:**
1. 로그인 상태 확인
2. 팝업 차단 해제
3. Safari의 경우: 설정 > Safari > 팝업 차단 끄기

### Q5. 검색 결과가 없어요

**A:**
1. 검색어 철자 확인 (예: "마커스" vs "마커스워십")
2. AI 검색 기능 활용
3. 필터 초기화 후 다시 검색
4. 원하는 곡이 없으면 직접 업로드해주세요!

### Q6. 곡을 업로드했는데 검색에 안 나와요

**A:** 업로드된 곡은 관리자 승인 후 검색에 노출됩니다.
승인까지 1~2일 정도 소요될 수 있습니다.

### Q7. 다른 팀원이 만든 콘티가 안 보여요

**A:**
1. 해당 팀에 가입되어 있는지 확인
2. 팀 페이지 > 콘티 탭에서 확인
3. 개인 콘티와 팀 콘티가 다른 메뉴입니다

### Q8. 비밀번호를 잊어버렸어요

**A:**
1. 로그인 화면에서 **[비밀번호 찾기]** 클릭
2. 가입 시 사용한 이메일 입력
3. 이메일로 받은 링크에서 재설정

Google 계정으로 가입했다면 Google 로그인을 사용하세요.
