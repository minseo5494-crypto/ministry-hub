## 2. 비용 구조

```
총 매출
  │
  ├── 결제 대행 수수료: 약 3.5% (플랫폼 부담, 파트너 청구 없음)
  │
  └── 순수익
       │
       ├── 40% → 플랫폼 운영
       │         • 서버/인프라 (Vercel, Supabase, CDN)
       │         • 개발 인건비
       │         • 마케팅/광고
       │         • 고객지원
       │         • 법률/회계
       │
       └── 60% → 사역 수익 풀 (Royalty Pool)
                 • 사역팀 정산
                 • 채보자 정산
                 • 편곡자 정산
```

**파트너에게 청구하지 않는 것:**
- 운영 수수료 ❌
- 관리비 ❌
- 입점비 ❌
- MG(최소보장금) ❌
- 결제 수수료 ❌ (플랫폼 부담)

## 3. 사역 수익 풀 (Royalty Pool) 상세

### 3-1. 풀 비율

```
전체 순수익의 60% = 사역 수익 풀
전체 순수익의 40% = 플랫폼 운영
```

### 3-2. 정산 지수 (Settlement Index, SI)

풀 내 배분은 정산 지수(SI)에 따라 결정된다.
SI는 단순 조회 수가 아닌 실제 예배 사용에 가까운 행동에 높은 가중치를 부여한다.

| 행동 | 설명 | 가중치 |
|------|------|--------|
| 예배 실행 | 예배일 기준 콘티 실행 (스트리밍) | **High** |
| 콘티 포함 | 해당 악보가 콘티에 등록됨 | **High** |
| 악보 다운로드 | PDF/이미지 다운로드 | Medium |
| 필기 및 주석 | 필기 도구 사용, 메모 추가 | Medium |
| 악보 열람 | 악보 상세 페이지 조회 | Low |

보조 지표:
- 좋아요(선호도): 단독 정산 X, 보정 계수로만 반영

**SI 산정 방식:**
- 상대 평가: 전체 SI 총합 대비 해당 팀 콘텐츠의 점유율
- 예: 전체 SI 합계가 10,000이고 기프티드 곡의 SI 합계가 1,500이면 → SI 점유율 15%

### 3-3. 배분율

```
┌──────────────────────────────────────────────────┐
│  사역 수익 풀 (순수익의 60%)                        │
│                                                    │
│    기본 배분율: 70%                                 │
│    ─────────────────                               │
│    풀의 70%가 SI 점유율에 따라 파트너에게 배분         │
│                                                    │
│    조건부 상향: 최대 90%                             │
│    ─────────────────                               │
│    아래 조건 충족 시 배분율 상향 가능                  │
│    ① 신곡/편곡을 WORSHEEP에서 최초 또는 우선 공개     │
│    ② 팀이 직접 검수한 공식 악보 제공                  │
│    ③ 플랫폼 마케팅/홍보 활동 참여                    │
│                                                    │
│    ※ 상향 조건은 선택적. 미참여 시에도 기본 정산 유지  │
│                                                    │
│    풀 내 미배분 잔여 (10~30%)                        │
│    ─────────────────                               │
│    플랫폼 풀 관리, 예비비, 인큐베이팅 등에 활용         │
└──────────────────────────────────────────────────┘
```

### 3-4. 팀 월수익 산식

```
팀 월수익 = 월 순수익 × 60%(풀) × SI 점유율 × 배분율(70~90%)
```

**예시 (3년차 MRR ₩5,140만 기준):**
- SI 점유율 15%, 기본 배분율 70%
- ₩5,140만 × 0.6 × 0.15 × 0.7 = **₩324만/월**

- 동일 조건에서 조건부 상향 90% 적용 시
- ₩5,140만 × 0.6 × 0.15 × 0.9 = **₩416만/월**

### 3-5. 편곡 악보 수익 분배

사용자가 기존 곡을 편곡하여 올린 경우:

```
편곡 악보 수익
  ├── 원곡자: 60% 이상
  └── 편곡자: 나머지

※ 편곡 악보가 많아질수록 원곡자의 총 수익도 증가
※ Founding Partner 원곡자 몫: 70% (우대)
```

## 4. 파트너 레벨별 수익 비교

### 4-1. 일반 파트너 (베타 파트너 MoU)

| 항목 | 내용 |
|------|------|
| 풀 비율 | 순수익의 60% |
| 기본 배분율 | **70%** |
| 조건부 상향 | 최대 **90%** |
| 편곡 원곡자 몫 | 60% |
| 정산 주기 | 분기별 |
| 수수료 | 없음 (결제 3.5%는 플랫폼 부담) |

### 4-2. Founding Partner (기프티드 Level 2)

| 항목 | 일반 | Founding |
|------|------|----------|
| 풀 비율 | 60% | 60% (동일) |
| 베타 기간 배분율 | - (미정산) | **100%** (수수료 0%) |
| 정식 후 12개월 배분율 | 70% | **90%** (수수료 감면) |
| 13개월 이후 | 70%~90% | 70%~90% (동일) |
| 편곡 원곡자 몫 | 60% | **70%** |
| 정산 주기 | 분기 | **월간 리포트 + 분기 정산** |

### 4-3. Co-Business Partner (기프티드 Level 3)

| 항목 | 내용 |
|------|------|
| SI 정산 | Founding Partner와 동일 (배분율 우대) |
| **Advisory Fee** | **월 순매출의 1~3%** (풀과 별개, 총매출 기반) |
| 조건 | 분기 1회 미팅 + 연 2회 공동 마케팅 |

**Advisory Fee 핵심:**
- SI와 무관하게 "플랫폼 전체 매출"에 연동
- 다른 팀이 늘어나도 Advisory 수익이 희석되지 않음
- SI + Advisory Fee 이중 수익 구조

## 5. 정산 방식

| 항목 | 내용 |
|------|------|
| 집계 | 매월 1일~말일 데이터 기준 |
| 정산 | 분기별 (3·6·9·12월 말일 기준) |
| 지급일 | 집계 마감 익월 15일 이내 |
| 최소 정산액 | ₩10,000 이상 시 지급 |
| 미달 시 | 다음 분기로 이월 |
| 지급 방법 | 파트너 지정 국내 은행 계좌 입금 |
| 베타 기간 | 유료 사용자 없으면 ₩0, 유료 전환 시 즉시 정산 개시 |

### 6-0. 구독 매출 산출 근거

가격 기준: 연 구독 60% / 월 구독 40% 가중 평균 (blend)

| 구분 | 월 구독 | 연 구독 (월 환산) | Blend 평균 |
|------|--------|-----------------|-----------|
| 개인 Pro | ₩4,900 | ₩4,083 | ₩4,410 |
| 팀 Basic (5명) | ₩14,900 | ₩12,417 | ₩13,410 |
| 팀 Pro (10명) | ₩29,900 | ₩24,917 | ₩26,910 |
| 교회 Small (~10팀) | ₩59,000 | ₩49,167 | ₩53,100 |
| 교회 Medium (~25팀) | ₩99,000 | ₩82,500 | ₩89,100 |
| 교회 Large (무제한) | ₩149,000 | ₩124,167 | ₩134,100 |

> 연 구독 = 월 구독 × 10개월 (약 17% 할인)
> 팀/교회 플랜: 1명이 구독하면 팀원 전원 이용 가능

단계별 MRR 산출:

