# WORSHEEP 빠른 시작 가이드

> 5분 안에 송폼이 표시된 악보를 다운로드해 보세요!

---

## 👋 WORSHEEP에 오신 것을 환영합니다

WORSHEEP는 교회 찬양팀을 위한 **올인원 악보 관리 플랫폼**입니다.

**이런 분들을 위해 만들었어요:**
- 매주 예배 콘티를 만드는 워십 리더
- 악보에 직접 필기하고 싶은 연주자
- 팀원들과 악보를 공유하고 싶은 찬양팀

---

## 1️⃣ 회원가입하기

### 웹사이트 접속
👉 [worsheep.com](https://worsheep.com) 접속

### 계정 만들기
1. 우측 상단 **[로그인]** 클릭
2. **[회원가입]** 선택
3. 이메일, 비밀번호, 이름 입력
4. 이메일 인증 완료

> 💡 **Tip:** Google 계정으로 간편 로그인도 가능해요!

---

## 2️⃣ 악보 찾기

### 기본 검색
1. 상단 검색창에 **곡 제목** 또는 **아티스트** 입력
2. Enter 키를 누르거나 검색 버튼 클릭

### 필터 사용하기
| 필터 | 설명 |
|------|------|
| **키(Key)** | C, D, E, F, G, A, B 선택 |
| **템포** | 느린/보통/빠른 |
| **테마** | 찬양, 경배, 기도, 치유 등 |
| **시즌** | 성탄절, 부활절, 추수감사절 |

### AI 검색 (Pro 기능)
자연어로 검색해 보세요!
- "부활절에 부르기 좋은 빠른 찬양"
- "위로가 되는 느린 곡"
- "시편 23편 관련 찬양"

---

## 3️⃣ 송폼 설정 & 다운로드

### Step 1. 곡 선택하기
1. 메인 페이지에서 원하는 곡의 **체크박스** 선택
2. 여러 곡을 선택할 수 있어요
3. 선택한 곡들이 하단에 표시됩니다

### Step 2. 송폼 설정하기
1. 선택한 곡을 클릭하면 **송폼 설정** 팝업이 열립니다
2. 예배에서 부를 순서를 설정해요:
   - **V** (Verse) - 절
   - **C** (Chorus) - 후렴
   - **B** (Bridge) - 브릿지
   - **I** (Intro) - 인트로
   - **O** (Outro) - 아웃트로
3. 예시: `I → V1 → C → V2 → C → B → C → O`
4. **[저장하기]** 클릭

### Step 3. 다운로드하기
1. **[다운로드]** 버튼 클릭
2. 다운로드 옵션 선택:
   | 옵션 | 설명 |
   |------|------|
   | **표지 포함** | 곡 목록이 포함된 표지 페이지 추가 |
   | **송폼 표시** | 악보 위에 송폼(V, C, B 등) 표시 |
3. 원하는 **파일명** 입력
4. **다운로드 형식** 선택 (PDF, 이미지)

### Step 4. 송폼 스타일 조정 (선택)
송폼을 설정한 경우, 스타일 조정 페이지가 나타납니다:
- **송폼 크기** 조절 (작게/보통/크게)
- **파트 태그** 크기 조절
- **색상** 변경

설정 완료 후 **[다운로드]** 클릭!

### Step 5. 공유하기
다운로드한 파일을 팀원들과 자유롭게 공유하세요!
- 카카오톡, 이메일 등으로 전송
- 클라우드에 업로드 후 링크 공유

> 💡 **Tip:** 매주 같은 형식으로 다운로드하면 팀원들이 익숙해져서 좋아요!

---

## 4️⃣ 팀 만들기

콘티를 저장하려면 먼저 팀이 필요해요!

### 팀 생성
1. 상단 메뉴에서 **[My-team]** 클릭
2. **[+ 새 팀 만들기]** 클릭
3. 팀 이름 입력 후 생성

### 팀원 초대하기
1. 팀 페이지에서 상단에 **초대 코드** 복사
3. 팀원에게 코드 전달

### 팀원이 가입하는 방법
1. **[팀 가입]** 메뉴 클릭
2. 초대 코드 입력
3. 가입 신청 (리더 승인 후 완료)

---

## 5️⃣ 콘티(예배 순서) 저장하기

콘티로 저장하면 나중에 다시 불러올 수 있어요!

### Step 1. 곡 선택 & 송폼 설정
위 3️⃣ 과정과 동일하게 곡 선택 및 송폼 설정

### Step 2. 콘티 저장
1. **[콘티 저장]** 버튼 클릭
2. 저장할 **팀** 선택
3. **콘티 이름** 입력 (예: "2월 첫째주 주일예배")
4. 완료!

### 저장된 콘티 확인하기
1. **[내 팀]** → 팀 선택 → **[콘티]** 탭
2. 저장된 콘티 목록에서 선택
3. 곡 순서 변경, 곡 추가/삭제 가능

---

## 6️⃣ 악보에 필기하기
                                                                                                 
### 악보에디터진입방법                                                            
                                                                                                 
**메인 페이지에서:**                             
  1. 원하는 곡의 **악보 에디터 버튼** (PPT 아이콘 📊) 클릭
  2. 악보 에디터가 열리면 필기 가능

  **콘티 목록에서:**
  1. **[내 팀]** → 팀 선택 → **[콘티]** 탭
  2. 원하는 콘티 오른쪽의 **악보 버튼** (PPT 아이콘 📊) 클릭
  3. 해당 콘티의 모든 악보가 에디터에 로드됩니다

### 필기 도구
| 도구 | 용도 |
|------|------|
| ✏️ **펜** | 자유롭게 그리기 |
| 🖍️ **형광펜** | 중요 부분 강조 |
| 🧹 **지우개** | 필기 삭제 |
| 📝 **텍스트** | 글자 입력 |
| ⭕ **올가미** | 여러 객체 선택/이동 |

### 필기 저장
- 필기 완료 후 **저장 버튼**을 눌러 저장하세요
- 저장하지 않고 나가면 변경 내용이 사라집니다
