# WORSHEEP 베타 테스터 가이드

**버전 1.0 | 2026년 1월**

---

## 환영합니다!

WORSHEEP 베타 테스트에 참여해 주셔서 감사합니다.

여러분의 피드백이 더 나은 서비스를 만드는 데 큰 도움이 됩니다.
이 가이드는 베타 테스트 기간 동안 서비스 사용법과 피드백 방법을 안내합니다.

## 1. 베타 테스트 안내

### 1.1 테스트 기간

- **시작**: 2026년 1월
- **종료**: 2026년 6월 (예정)
- **정식 출시**: 2026년 하반기 예정

### 1.2 베타 버전 특징

| 항목 | 설명 |
|------|------|
| **모든 기능 무료** | 베타 기간 동안 모든 기능을 무료로 이용 가능 |
| **테스트용 악보** | 인기 CCM 1,700곡+ 등록 (정식 출시 시 변경될 수 있음) |
| **버그 가능성** | 일부 기능에서 오류가 발생할 수 있음 |
| **데이터 보존** | 베타 기간 데이터는 정식 서비스로 이전됨 |

### 1.3 베타 테스트 주의사항

> **중요: 서비스 링크 공유를 삼가해 주세요.**

베타 기간 동안 서비스 링크를 외부에 공유하지 말아 주시기 바랍니다.

**이유:**
- 현재 등록된 악보들은 **저작권 허락 절차가 진행 중**입니다
- 정식 출시 전까지 **비공개 테스트** 형태로 운영됩니다
- 외부 유출 시 저작권 문제가 발생할 수 있습니다

**가능한 것:**
- 팀원 초대 (초대 코드 공유)
- 팀 내 콘티 공유

베타 테스트에 추가로 참여하고 싶은 분이 계시다면, 담당자에게 연락해 주세요.
정식 출시 후에는 자유롭게 공유하실 수 있습니다. 양해 부탁드립니다.

### 1.4 권장 환경

| 환경 | 권장 사항 |
|------|----------|
| **PC/노트북** | Chrome, Safari, Edge 최신 버전 (권장) |
| **태블릿** | iPad Safari, Android Chrome |
| **모바일** | 기본 기능 사용 가능 (일부 기능 제한) |

> **권장**: 악보 보기, 필기 기능은 **PC 또는 태블릿**에서 사용하시는 것을 권장합니다.
**중요** : 카카오톡 링크로 들어갈 경우 악보 다운로드가 정상적으로 되지 않을 수 있으니 크롬이나 PWA(아래 시작하기 단계 참고)로 사용 부탁드립니다.

## 2. 시작하기

### 2.1 회원가입

1. https://worsheep.org 접속
2. "회원가입" 클릭
3. 이메일 또는 Google 계정으로 가입
4. 이용약관 및 저작권 정책 동의
5. 이메일 인증 완료

### 2.2 홈 화면에 앱 추가 (PWA)

WORSHEEP을 앱처럼 사용할 수 있습니다. 홈 화면에 추가하면 브라우저 없이 바로 실행됩니다.

#### iPhone / iPad (Safari)
1. Safari에서 https://worsheep.org 접속
2. 하단 **공유 버튼** (사각형 + 화살표) 탭
3. **"홈 화면에 추가"** 선택
4. 이름 확인 후 **"추가"** 탭

#### Android (Chrome)
1. Chrome에서 https://worsheep.org 접속
2. 우측 상단 **점 3개 메뉴** 탭
3. **"홈 화면에 추가"** 또는 **"앱 설치"** 선택
4. **"설치"** 또는 **"추가"** 탭

#### PC (Chrome, Edge)
1. 브라우저에서 https://worsheep.org 접속
2. 주소창 오른쪽 **설치 아이콘** 클릭 (모니터 + 화살표 모양)
3. **"설치"** 클릭

> **장점**: 앱처럼 전체 화면으로 사용 가능하고, 홈 화면에서 바로 실행할 수 있습니다.

### 2.3 팀 생성/참여

#### 새 팀 만들기
1. 로그인 후 "My-team" 메뉴
2. "새 팀 만들기" 클릭
3. 팀 이름 입력
4. 초대 코드 생성 → 팀원에게 공유

#### 기존 팀 참여하기
1. 팀 리더에게 초대 코드 받기
2. "팀 관리" → "팀 참여하기"
3. 초대 코드 입력

## 5. 피드백 보내기

### 5.1 피드백 종류

| 종류 | 설명 | 우선순위 |
|------|------|----------|
| **버그 신고** | 오류, 에러 발생 | 높음 |
| **기능 제안** | 새로운 기능 아이디어 | 중간 |
| **불편 사항** | 사용하기 어려운 점 | 중간 |
| **칭찬/격려** | 좋았던 점 | 감사합니다! |

### 5.2 피드백 방법

**방법 1: 서비스 내 피드백**
- 우측 하단 "피드백" 버튼
- 내용 작성 후 전송

**방법 2: 이메일**
- minseo@worsheep.org
- 제목에 [베타피드백] 포함


### 5.3 좋은 피드백 예시

**버그 신고**
```
[버그] 콘티에서 곡 순서 변경 안됨

- 환경: iPhone 15, Safari
- 상황: 콘티 편집 화면에서 드래그로 곡 순서 변경 시도
- 증상: 드래그는 되는데 놓으면 원래 자리로 돌아감
- 재현: 항상 발생
```

**기능 제안**
```
[제안] 악보에 메트로놈 기능 추가

- 악보 보면서 연습할 때 메트로놈이 있으면 좋겠어요
- BPM 설정하고 재생하는 간단한 기능이면 충분할 것 같습니다
```

## 7. 자주 묻는 질문

### Q: 베타 기간에 입력한 데이터는 어떻게 되나요?
**A:** 정식 서비스로 그대로 이전됩니다. 콘티, 필기, 팀 정보 모두 유지됩니다.

### Q: 베타 기간에 등록된 악보가 정식 버전에서 사라질 수 있나요?
**A:** 네, 저작권 허락을 받지 못한 곡은 정식 출시 시 삭제될 수 있습니다.

### Q: 버그를 발견하면 어떻게 하나요?
**A:** 피드백 기능 또는 이메일로 알려주세요. 스크린샷이 있으면 더 좋습니다.

### Q: 친구도 베타 테스트에 참여할 수 있나요?
**A:** 현재는 초대된 분들만 참여 가능합니다. 추가 초대가 필요하시면 연락주세요.
