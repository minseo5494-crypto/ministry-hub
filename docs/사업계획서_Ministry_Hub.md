# Ministry Hub 사업계획서

## 내부용 | 2024년 12월

---

## 1. Executive Summary

### 서비스 개요
**Ministry Hub**는 교회 찬양팀을 위한 올인원 악보 관리 및 콘티 제작 플랫폼입니다. 찬양팀이 예배를 준비하는 전 과정을 디지털화하여 효율성을 극대화합니다.

### 핵심 가치 제안
- **시간 절약**: 악보 검색, 정리, 공유에 드는 시간을 80% 절감
- **협업 강화**: 팀원 간 실시간 악보 공유 및 필기 동기화
- **전문성 향상**: 송폼, 파트 태그 등 전문 기능으로 찬양 퀄리티 향상

### 현재 상태
- 개발 단계: MVP 완료, 내부 테스트 중
- 팀 구성: 2-3인 소규모 팀
- 수익 모델: 프리미엄 + 구독형 하이브리드

---

## 2. 문제 정의 (Problem)

### 찬양팀이 겪는 현실적 문제들

| 문제 | 현재 상황 | 영향 |
|------|----------|------|
| **악보 관리** | 카카오톡, 드라이브, USB에 분산 저장 | 필요할 때 악보를 찾지 못함 |
| **콘티 작성** | 수기 또는 엑셀로 작성 | 변경 시 재작성 필요, 공유 어려움 |
| **악보 필기** | 종이에 직접 필기 또는 PDF 앱 사용 | 악보별 필기 관리 불가 |
| **팀 공유** | 카톡으로 파일 전송 | 버전 관리 불가, 파일 만료 |
| **연습/예배 시** | 여러 앱 번갈아 사용 | 집중력 저하, 시간 낭비 |

### 타겟 시장 규모

```
한국 교회 수: 약 60,000개
찬양팀 보유 교회: 약 40,000개 (67%)
팀당 평균 인원: 8-15명
잠재 사용자: 약 400,000 ~ 600,000명
```

## 3. 솔루션 (Solution)

### 3.1 현재 기능 (MVP)

#### 악보 관리
- 악보 업로드 (PDF, 이미지)
- 곡 정보 관리 (제목, 아티스트, 키, BPM)
- 검색 및 필터링
- 즐겨찾기 기능

#### 콘티 제작
- 드래그 앤 드롭으로 곡 순서 편집
- 콘티 저장 및 불러오기
- 팀별 콘티 관리

#### 악보 필기 (핵심 기능)
- 펜, 형광펜, 지우개 도구
- 송폼 오버레이 (Intro, Verse, Chorus 등)
- 파트 태그 추가
- 멀티 페이지 PDF 지원
- 필기 내보내기 (PDF/PNG)
- 터치 스와이프 페이지 넘김
- 핀치 줌 지원

#### 팀 관리
- 팀 생성 및 멤버 초대
- 팀별 악보 라이브러리
- 공유 콘티

### 3.2 기술 스택

```
Frontend: Next.js 15, React, TypeScript, Tailwind CSS
Backend: Supabase (Auth, Database, Storage)
배포: Vercel
기타: PDF.js, jsPDF, perfect-freehand
```

## 4. 수익 모델

### 4.1 프리미엄 + 구독 하이브리드 모델

#### Free 플랜
- 개인 악보 저장 30곡
- 필기 기능 (기본)
- 1개 팀 참여

#### Pro 플랜 (월 4,900원)
- 무제한 악보 저장
- 고급 필기 도구 (파트 태그, 송폼 커스터마이징)
- 필기 내보내기 (PDF/PNG)
- 3개 팀 참여
- 클라우드 동기화

#### Team 플랜 (월 19,900원, 팀당)
- 팀 전체 Pro 기능
- 팀 악보 라이브러리 무제한
- 팀원 최대 20명
- 관리자 대시보드
- 콘티 히스토리


### Phase 2: 전용 앱 개발 (6개월 ~ 12개월)

#### 모바일 앱 (iOS/Android)
- React Native 또는 Flutter 기반
- 오프라인 악보 저장
- Apple Pencil / S Pen 최적화
- 예배 모드 (화면 꺼짐 방지, 자동 페이지 넘김)

#### 태블릿 전용 앱
- iPad Pro / Galaxy Tab 최적화
- 악보 필기 고급 기능
- 멀티태스킹 (악보 + 가사 동시 표시)

### Phase 3: CCM 스트리밍 서비스 (12개월 ~ 18개월)

#### 유튜브 연동 전략
```
1. 유튜브 CCM 영상 큐레이션
   - CCM 아티스트 공식 채널 연동
   - 플레이리스트 생성/공유

2. 악보 + 영상 동기화
   - 악보 보면서 유튜브 영상 재생
   - 곡 검색 시 유튜브 영상 자동 연결

3. 예배 프리뷰 모드
   - 콘티 순서대로 영상 자동 재생
   - 팀 연습용 플레이리스트 생성
```

#### 향후 음원 제휴 가능성
- 1단계: 유튜브 연동으로 사용자 확보
- 2단계: CCM 레이블과 협의 (예: 예수전도단, 마커스 등)
- 3단계: 자체 스트리밍 라이선스 확보

### Phase 4: 플랫폼 확장 (18개월 이후)

#### 악보 마켓플레이스
- 개인/교회가 편곡 악보 판매
- 수수료 모델 (15~20%)
- 공식 악보 제휴 (소향뮤직, 하이스코어 등)

#### AI 기능
- 코드 자동 인식
- 키 변환 자동화
- 악보 스캔 → 디지털 변환
- 송폼 자동 제안

#### 커뮤니티
- 찬양팀 네트워크
- Q&A 게시판
- 편곡 공유

## 7. 경쟁 분석

### 직접 경쟁자

| 서비스 | 강점 | 약점 | Ministry Hub 차별점 |
|--------|------|------|-------------------|
| **Planning Center** | 글로벌 표준, 풍부한 기능 | 영어 전용, 높은 가격 | 한국어, 합리적 가격 |
| **GoodNotes/Notability** | 뛰어난 필기 | 악보 특화 X | 악보 전용 기능 |
| **구글 드라이브** | 무료, 익숙함 | 악보 관리 기능 없음 | 악보 특화 UX |

### 경쟁 우위
1. **한국 CCM 시장 특화**: 한글 송폼, 한국 아티스트 데이터
2. **올인원 솔루션**: 악보 + 콘티 + 필기 + 팀 관리
3. **합리적 가격**: Planning Center 대비 1/3 가격
4. **모바일 최적화**: 태블릿 필기에 최적화된 UX
