악보 찾느라 카톡 뒤지고, 드라이브 뒤지고, 블로그 검색하시나요?
콘티 만들 때마다 엑셀 열고, PDF 합치고, 카톡으로 공유하시나요?

## 핵심 기능

### 1. 통합 악보 검색
- **1,000곡+ CCM 악보** 한 곳에서 검색
- 곡명, 아티스트, 가사, Key, 박자, 템포로 필터링
- **AI 자연어 검색**: "부활절에 부르기 좋은 빠른 찬양"

### 2. 콘티 제작 & 공유
- 드래그 앤 드롭으로 **30초 만에 콘티 완성**
- 팀원과 **실시간 공유**
- 지난 콘티 히스토리 관리

### 3. 악보 필기
- 펜, 형광펜, 텍스트로 **악보에 직접 필기**
- **송폼 표시**: V, C, B 한눈에
- **파트 태그**: Vocal, Bass, Drum 역할 표시

### 4. 팀 협업
- 팀 생성 & 초대 코드로 간편 참여
- 팀 악보 라이브러리 공유
- 역할별 권한 관리

### 5. 다운로드
- PDF, PPT, 이미지로 내보내기
- 콘티 전체 악보 **일괄 다운로드**
