## 1. WORSHEEP란?

WORSHEEP는 **교회 찬양팀을 위한 올인원 악보 관리 플랫폼**입니다.

### 이런 분들에게 추천합니다

- 악보가 카톡, 드라이브, 이메일에 흩어져 있어 찾기 힘든 분
- 매주 엑셀로 콘티(예배 순서) 작성이 번거로운 분
- 아이패드에서 악보에 필기하고 싶은 분
- 팀원들과 악보를 쉽게 공유하고 싶은 분

### 주요 기능

| 기능 | 설명 |
|------|------|
| **악보 검색** | 1,700곡+ 악보를 절기, 테마, 조성 등으로 검색 |
| **콘티 작성** | 드래그앤드롭으로 쉽게 예배 순서 작성 |
| **팀 협업** | 팀원들과 콘티, 악보 실시간 공유 |
| **악보 필기** | 아이패드에서 악보에 직접 필기 |
| **다운로드** | PDF, PPT, 이미지로 악보 다운로드 |
| **AI 검색** | "크리스마스 느린 찬양" 같은 자연어 검색 |

## 2. 시작하기

### 2.1 회원가입

1. **worsheep.com** 접속 (또는 앱 열기)
2. **"회원가입"** 버튼 클릭
3. 이메일, 비밀번호 입력 (또는 Google 계정으로 가입)
4. 교회명 입력 (선택사항)
5. **"가입하기"** 클릭

### 2.2 로그인

1. 이메일과 비밀번호 입력
2. **"로그인"** 클릭
3. 또는 **"Google로 로그인"** 클릭

### 2.3 아이패드/모바일에서 앱처럼 사용하기

WORSHEEP는 **PWA(웹앱)**로, 홈 화면에 추가하면 앱처럼 사용할 수 있습니다.

**iPad/iPhone:**
1. Safari에서 worsheep.com 접속
2. 하단 **공유 버튼** (⬆️) 탭
3. **"홈 화면에 추가"** 선택
4. **"추가"** 탭

**Android:**
1. Chrome에서 worsheep.com 접속
2. 우측 상단 **메뉴 (⋮)** 탭
3. **"홈 화면에 추가"** 선택

## 3. 메인 화면 - 곡 검색

로그인하면 메인 화면에서 모든 곡을 검색할 수 있습니다.

### 3.1 검색창 사용하기

```
┌─────────────────────────────────────┐
│ 🔍 곡명, 팀명으로 검색...            │
└─────────────────────────────────────┘
```

- 곡 이름 일부 입력 (예: "감사", "주님")
- 팀 이름으로 검색 (예: "마커스", "제이어스")
- **Enter** 또는 검색 버튼 클릭

### 3.2 AI 자연어 검색

검색창 위의 **"AI 자연어 검색"** 버튼을 켜면 자연어로 검색할 수 있습니다.

**예시:**
- "크리스마스에 부르기 좋은 찬양"
- "빠른 템포의 감사 찬양"
- "차분한 D키 찬양"

AI가 검색어를 분석하여 자동으로 필터를 적용합니다.

### 3.3 필터 사용하기

왼쪽 **필터** 패널에서 조건을 선택하세요:

| 필터 | 설명 | 예시 |
|------|------|------|
| **절기** | 특정 시즌 곡 | 크리스마스, 부활절, 추수감사절 |
| **테마** | 곡의 주제 | 감사, 찬양, 은혜, 헌신 등 (다중 선택) |
| **조성** | 곡의 키 | C, D, E, Am, Bm 등 |
| **박자** | 박자 | 4/4, 3/4, 6/8 등 |
| **템포** | 빠르기 | 느림, 보통, 빠름 |
| **BPM** | 정확한 빠르기 | 60~80, 100~120 등 |

**필터 초기화**: 필터 상단의 "초기화" 버튼 클릭

### 3.4 곡 카드 정보

각 곡 카드에서 확인할 수 있는 정보:

```
┌─────────────────────────────────┐
│ 🎵 감사와 찬양 드리며            │
│ 마커스워십                       │
│ ───────────────────────────────│
│ 키: D  |  BPM: 72  |  4/4      │
│ 템포: 느림  |  테마: 감사        │
│ ───────────────────────────────│
│ [악보보기] [❤️ 좋아요] [▶️ YouTube] │
└─────────────────────────────────┘
```

### 3.5 뷰 모드 변경

우측 상단에서 뷰 모드를 선택할 수 있습니다:
- **그리드 뷰** (⬜⬜): 카드 형태로 보기
- **리스트 뷰** (☰): 목록 형태로 보기

### 3.6 정렬 옵션

- **최신순**: 최근 추가된 곡 먼저
- **인기순**: 좋아요 많은 곡 먼저
- **이름순**: 가나다 순

## 4. 악보 보기 & 다운로드

### 4.1 악보 미리보기

1. 곡 카드에서 **"악보보기"** 클릭
2. 악보가 모달/팝업으로 표시됨
3. 여러 페이지가 있으면 좌우 화살표로 이동

### 4.2 악보 다운로드

**단일 곡 다운로드:**
1. 악보 미리보기에서 **다운로드 버튼** 클릭
2. 형식 선택: PDF, 이미지
3. 다운로드 시작

**여러 곡 한번에 다운로드:**
1. 곡 카드 좌측 **체크박스** 선택
2. 하단에 나타나는 **"선택한 곡 다운로드"** 버튼 클릭
3. 형식 선택 (PDF, PPT)
4. **송폼 위치 설정** (선택사항)
   - 곡 제목 위치, 크기, 색상 조정
5. 다운로드 시작

### 4.3 송폼(곡 제목) 설정

다운로드 시 악보에 곡 제목을 표시할 수 있습니다:

- **위치**: 상단/하단/좌측/우측
- **크기**: 폰트 크기 조정
- **색상**: 글자 색상 선택
- **투명도**: 배경 투명도 조정

## 5. 팀 만들기 & 참여하기

팀을 만들면 팀원들과 콘티, 악보를 공유할 수 있습니다.

### 5.1 새 팀 만들기

1. 좌측 메뉴에서 **"내 팀"** 클릭
2. **"새 팀 만들기"** 버튼 클릭
3. 팀 정보 입력:
   - 팀 이름 (예: "○○교회 찬양팀")
   - 팀 타입 (교회 내부/외부)
   - 교회명
4. **"만들기"** 클릭
5. 자동으로 **초대 코드**가 생성됩니다

### 5.2 팀원 초대하기

**방법 1: 초대 코드 공유**
1. 팀 페이지에서 **"팀원 초대"** 클릭
2. 표시된 **초대 코드** 복사
3. 카톡/이메일로 팀원에게 전달

**방법 2: 초대 링크 공유**
1. **"초대 링크 복사"** 클릭
2. 링크를 팀원에게 전달

### 5.3 팀에 참여하기

1. 좌측 메뉴에서 **"팀 참여"** 클릭
2. 받은 **초대 코드** 입력
3. **"참여하기"** 클릭
4. 팀 목록에 추가됩니다

### 5.4 팀 권한 관리

팀장(리더)은 팀원의 권한을 관리할 수 있습니다.

**팀 설정 > 멤버 관리:**
- 멤버 역할 변경 (리더, 부리더, 팀원)
- 멤버 제거

**팀 설정 > 권한 관리:**
- 커스텀 직책 생성 (예: "콘티 담당자")
- 직책별 권한 설정:
  - 콘티 보기/생성/편집/삭제
  - 악보 보기/다운로드
  - 고정곡 관리
  - 멤버/권한 관리

## 6. 콘티(셋리스트) 만들기

콘티는 예배 때 부를 곡 순서입니다.

### 6.1 새 콘티 만들기

1. **내 팀** > 팀 선택
2. **"새 콘티 만들기"** 버튼 클릭
3. 콘티 정보 입력:
   - 제목 (예: "2025년 1월 5일 주일예배")
   - 날짜
   - 메모 (선택)
4. **"만들기"** 클릭

### 6.2 콘티에 곡 추가하기

**방법 1: 곡 검색에서 추가**
1. 메인 화면에서 곡 검색
2. 추가하고 싶은 곡의 **체크박스** 선택
3. 하단 **"콘티에 추가"** 버튼 클릭
4. 콘티 선택
5. **"추가"** 클릭

**방법 2: 콘티 편집에서 추가**
1. 콘티 상세 페이지에서 **"곡 추가"** 클릭
2. 곡 검색
3. 원하는 곡 선택
4. **"추가"** 클릭

### 6.3 콘티 곡 순서 변경

1. 콘티 편집 모드에서
2. 곡을 **드래그앤드롭**으로 순서 변경
3. 자동 저장됨

### 6.4 곡별 악보 형태 선택

각 곡마다 다른 악보 형태(키)를 선택할 수 있습니다:

1. 콘티에서 곡 클릭
2. **"악보 형태"** 선택
3. 원하는 버전 선택 (예: "원키", "E키", "밴드악보")

### 6.5 고정곡 설정

매번 콘티에 포함되는 곡을 미리 설정할 수 있습니다:

1. 팀 페이지에서 **"고정곡 관리"** 클릭
2. **"고정곡 추가"** 클릭
3. 곡 검색 후 선택
4. 위치 설정 (시작/끝)
5. 새 콘티 생성 시 자동으로 포함됩니다

### 6.6 콘티 다운로드

1. 콘티 상세 페이지에서 **"다운로드"** 클릭
2. 형식 선택 (PDF, PPT)
3. 옵션 설정:
   - 송폼 포함 여부
   - 악보 순서
4. **"다운로드"** 클릭

## 7. 악보에 필기하기

아이패드나 태블릿에서 악보에 직접 필기할 수 있습니다.

### 7.1 악보 에디터 열기

1. 곡 상세에서 **"필기하기"** 또는 **"악보 편집"** 클릭
2. 악보 에디터가 열립니다

### 7.2 필기 도구

에디터 상단 툴바에서 도구를 선택하세요:

| 도구 | 아이콘 | 설명 |
|------|--------|------|
| **펜** | ✏️ | 자유롭게 그리기 |
| **형광펜** | 🖍️ | 강조 표시 (반투명) |
| **지우개** | 🧹 | 그린 것 지우기 |
| **텍스트** | T | 텍스트 추가 |
| **팬(이동)** | ✋ | 화면 이동 |
| **라소(선택)** | ⭕ | 영역 선택 |

### 7.3 펜 설정

펜/형광펜 선택 후:
- **색상**: 팔레트에서 색상 선택
- **굵기**: 슬라이더로 굵기 조정

### 7.4 페이지 이동

여러 페이지 악보의 경우:
- **◀️ / ▶️** 버튼으로 페이지 이동
- 또는 하단 **페이지 번호** 클릭

### 7.5 필기 저장

- **자동 저장**: 필기 내용이 자동으로 저장됩니다
- **수동 저장**: 상단 **"저장"** 버튼 클릭

### 7.6 피아노 악보 추가

1. 툴바에서 **"피아노 악보"** 탭 선택
2. 마디 추가
3. 음표/코드 입력:
   - 음표: C4, D4, E4 등 클릭
   - 코드: Bb7, G, Am 등 입력
4. 자동으로 악보에 표시됩니다

### 7.7 드럼 악보 추가

1. 툴바에서 **"드럼 악보"** 탭 선택
2. 드럼 파트별 입력:
   - **HH**: 하이햇
   - **SN**: 스네어
   - **KK**: 킥(베이스드럼)
   - **TH/TM/TL**: 탐탐
   - **CY**: 심벌
   - **RD**: 라이드
3. 각 박자 위치에 클릭하여 음표 추가

### 7.8 송폼 배치

악보에 곡 제목을 표시할 수 있습니다:

1. 툴바에서 **"송폼"** 탭 선택
2. **"송폼 추가"** 클릭
3. 드래그하여 위치 조정
4. 스타일 설정:
   - 텍스트 (기본: 곡명)
   - 글꼴 크기
   - 색상
   - 배경색/투명도

### 7.9 파트 태그 추가

악보에 파트 표시를 추가할 수 있습니다:

1. **"파트 태그"** 탭 선택
2. 태그 선택 (싱어, 피아노, 기타, 드럼, 베이스 등)
3. 악보 위 원하는 위치에 클릭
4. 태그가 표시됩니다

## 8. 개인 콘티 뷰

팀 콘티를 자신만의 버전으로 커스터마이징할 수 있습니다.

### 8.1 개인 콘티 뷰란?

- 팀 콘티의 원본은 그대로 유지
- 나만의 필기 노트로 곡을 대체하거나 추가
- 다른 팀원에게는 영향 없음

### 8.2 곡을 내 필기로 대체하기

1. 콘티 상세 페이지에서 **"개인화"** 버튼 클릭
2. 대체하고 싶은 곡 옆 **"필기로 대체"** 클릭
3. 내 필기 노트 목록에서 선택
4. **"적용"** 클릭

### 8.3 필기 노트 끼워넣기

곡과 곡 사이에 필기 노트를 추가할 수 있습니다:

1. 개인화 모드에서
2. 곡 사이 **"+ 노트 끼워넣기"** 클릭
3. 필기 노트 선택
4. **"추가"** 클릭

### 8.4 개인화 초기화

원본 콘티로 돌아가려면:
1. **"개인화 초기화"** 클릭
2. 확인 후 원본 복원

## 9. 곡 업로드하기

자신만의 악보를 업로드할 수 있습니다.

### 9.1 악보 업로드 페이지 접근

1. 좌측 메뉴에서 **"곡 업로드"** 클릭
2. 또는 상단 **"+ 업로드"** 버튼 클릭

### 9.2 개별 곡 업로드

1. **"곡 정보"** 입력:
   - 곡명 (필수)
   - 팀명/아티스트
   - 조성, BPM, 박자, 템포
   - 테마, 절기
   - 가사
   - YouTube URL
2. **"악보 파일"** 업로드:
   - PDF 또는 이미지 파일
   - 여러 파일 선택 가능 (여러 페이지)
3. **"공개 범위"** 설정:
   - **공개**: 모든 사용자에게 공개
   - **팀 공개**: 내가 속한 팀에만 공개
   - **비공개**: 나만 볼 수 있음
4. **"업로드"** 클릭

### 9.3 엑셀로 대량 업로드

여러 곡을 한 번에 업로드할 수 있습니다:

1. **"엑셀 업로드"** 탭 선택
2. **템플릿 다운로드** 클릭
3. 엑셀 파일 작성:
   ```
   곡ID | 곡명 | 팀명 | 조성 | BPM | 박자 | 템포 | 가사 | 절기 | 테마 | YouTube URL | 공개범위
   ```
4. 작성한 엑셀 파일 업로드
5. **"업로드 시작"** 클릭

### 9.4 악보 파일 업로드

엑셀 업로드 후 악보 파일을 연결합니다:

1. **"악보 파일 업로드"** 탭
2. 파일 선택 (여러 파일 가능)
   - 파일명 규칙: `곡ID_페이지번호.pdf` 또는 `곡ID.pdf`
3. 자동으로 곡과 매칭됩니다
4. 진행률 표시

## 11. 자주 묻는 질문 (FAQ)

### Q1. 악보를 다운로드했는데 화질이 낮아요.
**A:** PDF 다운로드를 사용하면 원본 화질 그대로 받을 수 있습니다. 이미지 다운로드는 압축될 수 있습니다.

### Q2. 아이패드에서 Apple Pencil로 필기가 안 돼요.
**A:** Safari 브라우저를 사용하고, 화면 확대/축소를 하지 않은 상태에서 시도해보세요. 필기 도구(펜/형광펜)가 선택되어 있는지 확인하세요.

### Q3. 팀원이 초대 코드로 참여가 안 돼요.
**A:** 초대 코드는 대소문자를 구분합니다. 정확히 입력했는지 확인하세요. 문제가 지속되면 새 초대 코드를 생성해보세요.

### Q4. 업로드한 곡이 검색에 안 나와요.
**A:** 공개 범위가 "비공개"로 설정되어 있으면 다른 사람에게 보이지 않습니다. 마이페이지에서 공개 범위를 확인하세요.

### Q5. 콘티에서 곡 순서를 바꾸고 싶어요.
**A:** 콘티 편집 모드에서 곡을 드래그앤드롭하면 순서를 변경할 수 있습니다.

### Q6. 필기한 내용이 저장 안 됐어요.
**A:** 인터넷 연결 상태를 확인하세요. 필기 내용은 자동 저장되지만, 연결이 불안정하면 저장이 지연될 수 있습니다. 상단 "저장" 버튼을 눌러 수동 저장하세요.

### Q7. 팀에서 나가고 싶어요.
**A:** 팀 페이지 > 설정 > "팀 나가기"를 클릭하세요. 팀장인 경우 다른 팀원에게 팀장을 먼저 위임해야 합니다.

### Q8. 오프라인에서도 사용할 수 있나요?
**A:** 현재는 온라인 연결이 필요합니다. 미리 악보를 다운로드해두면 오프라인에서도 볼 수 있습니다.

### Q9. 곡 정보가 잘못되어 있어요. 수정할 수 있나요?
**A:** 본인이 업로드한 곡은 마이페이지에서 수정할 수 있습니다. 다른 사람이 업로드한 곡은 피드백 버튼으로 알려주세요.

### Q10. 특정 곡이 검색이 안 돼요.
**A:** AI 검색을 켜고 다양한 키워드로 검색해보세요. 또는 필터를 초기화하고 곡명 일부로 검색해보세요.
