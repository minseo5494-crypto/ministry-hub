# WORSHEEP 사업계획서

## 내부용 | 2026년 1월

---

## 1. Executive Summary

### 서비스 개요
**WORSHEEP**는 교회 찬양팀을 위한 올인원 악보 관리 및 콘티 제작 플랫폼입니다. 악보 관리, 콘티 제작, 악보 필기, 팀 협업을 하나의 서비스에서 제공하여 찬양팀의 예배 준비 과정을 혁신합니다.

### 핵심 가치 제안
| 가치 | 설명 |
|------|------|
| **시간 절약** | 악보 검색, 정리, 공유에 드는 시간을 80% 절감 |
| **협업 강화** | 팀원 간 실시간 악보/콘티 공유 및 동기화 |
| **전문성 향상** | 송폼, 파트 태그 등 찬양팀 전용 기능 |
| **올인원** | 분산된 도구들을 하나로 통합 |

### 현재 상태
| 항목 | 상태 |
|------|------|
| 개발 단계 | MVP 완료 (기능 완성도 95%) |
| 베타 테스트 | 2026년 1월 시작 예정 |
| 등록 곡 수 | 1,000곡+ |
| 팀 구성 | 1인 창업 (2월 디자이너 합류) |
| 사업자 등록 | 미등록 (개인 프로젝트) |

## 2. 문제 정의 (Problem)

### 찬양팀이 겪는 현실적 문제들

| 문제 | 현재 상황 | 영향 |
|------|----------|------|
| **악보 관리** | 카카오톡, 드라이브, USB에 분산 저장 | 필요할 때 악보를 찾지 못함 |
| **콘티 작성** | 수기 또는 엑셀로 작성 | 변경 시 재작성 필요, 공유 어려움 |
| **악보 필기** | 종이에 직접 필기 또는 PDF 앱 사용 | 악보별 필기 관리 불가, 송폼 표시 어려움 |
| **팀 공유** | 카톡으로 파일 전송 | 버전 관리 불가, 파일 만료 |
| **연습/예배** | 여러 앱 번갈아 사용 | 집중력 저하, 시간 낭비 |

### 타겟 시장 규모

#### 국내 시장
```
한국 교회 수: 약 60,000개
찬양팀 보유 교회: 약 40,000개 (67%)
팀당 평균 인원: 8-15명
잠재 사용자: 약 400,000 ~ 600,000명
```

#### 글로벌 시장 (해외 한인교회)
```
해외 한인교회 수: 약 5,000개+
주요 지역: 미국, 캐나다, 호주, 유럽
잠재 사용자: 약 50,000명+
```

## 3. 솔루션 (Solution)

### 3.1 현재 구현된 기능

#### 악보 관리 (100% 완료)
- PDF/이미지 악보 업로드
- 곡 정보 관리 (제목, 아티스트, 키, BPM, 템포)
- AI 자연어 검색 ("부활절에 부르기 좋은 빠른 찬양")
- 다양한 필터링 (테마, 시즌, 키, 템포)
- 즐겨찾기(좋아요) 기능

#### 콘티 제작 (100% 완료)
- 드래그 앤 드롭으로 곡 순서 편집
- 팀별 콘티 저장 및 관리
- 예배 유형/날짜 설정
- 고정곡 기능 (여는찬양, 마치는찬양 등)
- 콘티 복사 및 공유

#### 악보 필기 (100% 완료)
- 펜, 형광펜, 텍스트 도구
- 송폼 오버레이 (Intro, Verse, Chorus, Bridge 등)
- 파트 태그 추가 (Vocal, Bass, Drum 등)
- 멀티 페이지 PDF 지원
- 필기 내보내기 (PDF/PNG/PPT)
- 터치/Apple Pencil 지원

#### 팀 관리 (100% 완료)
- 팀 생성 및 초대 코드 발급
- 멤버 역할 관리 (리더, 관리자, 멤버)
- 팀별 악보 라이브러리
- 권한 기반 접근 제어

#### 기타 기능
- 관리자 대시보드 (통계, 콘텐츠 관리)
- Google OAuth 로그인
- 다운로드 기능 (PDF, PPT, 이미지)

## 4. 콘텐츠 및 저작권 모델

### 4.1 악보 콘텐츠 분류

| 분류 | 설명 | 저작권 처리 |
|------|------|-------------|
| **공식 악보** | 저작권자(팀/출판사)와 파트너십 체결 | 사용 허락 계약, 수익 50% 정산 |
| **퍼블릭 도메인** | 저작권 만료된 곡 (저작자 사후 70년) | 자유 이용 가능 |
| **사용자 업로드** | 사용자가 직접 등록한 악보 | 업로더 책임, 공개 범위 선택 가능 |

### 4.2 베타 vs 정식 서비스

| 구분 | 베타 (현재) | 정식 서비스 |
|------|-------------|-------------|
| **공식 악보** | 테스트용으로 인기곡 1,000곡+ 등록 | 사용 허락 받은 곡만 유지 |
| **퍼블릭 도메인** | 동일 | 동일 |
| **사용자 업로드** | 동일 | 동일 |

> **중요**: 정식 서비스 출시 시, 파트너십이 체결되지 않은 곡은 삭제되며 공식 악보와 퍼블릭 도메인 곡만 남습니다.

### 4.3 사용자 업로드 시스템

#### 업로드 공개 범위
| 범위 | 설명 | 용도 |
|------|------|------|
| **비공개** | 본인만 열람 가능 | 개인 악보 보관 |
| **팀 공개** | 소속 팀원만 열람 가능 | 팀에서 자주 쓰는 곡, 자체 편곡 |
| **전체 공개** | 모든 사용자 열람 가능 | 공유하고 싶은 악보 |

#### 저작권 책임
- 업로드 시 저작권 정책 동의 필수
- **저작권 책임은 업로더에게 있음** (회원가입/업로드 시 동의)
- 전체 공개 악보: 저작권 확인 불가하므로 신고 시 삭제

### 4.4 저작권 보호 정책 (Safe Harbor)

저작권법 제102조(온라인서비스제공자의 책임 제한)에 따라 다음 조건 충족 시 플랫폼 면책:

| 조건 | WORSHEEP 대응 |
|------|---------------|
| **저작권 침해 인지 시 즉시 조치** | 신고 접수 후 3영업일 내 삭제 |
| **신고-삭제 절차 마련** | 저작권 정책 페이지에 신고 절차 명시 |
| **반복 침해자 제재** | 3진 아웃제 (경고 → 30일 제한 → 영구 정지) |

#### 신고 처리 절차
```
신고 접수 (이메일) → 검토 (3영업일) → 콘텐츠 삭제 → 업로더 통보 → 이의신청 처리
```
