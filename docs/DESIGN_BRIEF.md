## 1. 서비스 개요

### Ministry Hub란?
**교회 찬양팀을 위한 올인원 악보 관리 플랫폼**입니다.

찬양팀이 겪는 문제:
- 악보가 카톡, 드라이브, 이메일에 흩어져 있음
- 매주 엑셀로 콘티(예배 순서) 작성이 번거로움
- 팀원들과 악보 공유가 불편함
- 아이패드에서 악보에 필기하기 어려움

Ministry Hub의 해결책:
- 모든 악보를 한 곳에서 검색/관리
- 드래그앤드롭으로 쉽게 콘티 작성
- 팀원들과 실시간 공유
- 아이패드에서 악보에 직접 필기

### 현재 상태
| 항목 | 현황 |
|------|------|
| 등록 곡 수 | 1,700+ |
| 개발 단계 | 베타 테스트 준비 중 |
| 주요 기능 | 모두 구현 완료 |
| 디자인 | 기능 중심으로 개발, 개선 필요 |

## 2. 타겟 사용자

### 주요 사용자
| 구분 | 특징 |
|------|------|
| **1차 타겟** | 찬양팀 리더 (인도자, 밴드 리더) |
| **2차 타겟** | 찬양팀 팀원 (보컬, 악기 연주자) |
| **연령대** | 20대 ~ 40대 |
| **IT 숙련도** | 중간 (카톡, 인스타 능숙) |

### 사용 시나리오
1. **주중**: 다음 주일 예배 콘티 작성, 악보 검색
2. **연습 시**: 아이패드로 악보 보면서 필기
3. **예배 중**: 아이패드로 콘티 순서대로 악보 보기

## 3. 사용 환경

### 디바이스 우선순위
```
1순위: 아이패드 (예배/연습 시 주로 사용)
2순위: 모바일 (이동 중 확인)
3순위: 데스크탑 (콘티 작성, 업로드)
```

### 브레이크포인트
| 디바이스 | 해상도 |
|---------|--------|
| 모바일 | 375px ~ 767px |
| 태블릿 | 768px ~ 1023px |
| 데스크탑 | 1024px ~ |

### 특수 고려사항
- **Apple Pencil 사용**: 악보에 필기 기능
- **PDF 뷰어**: 악보 확대/축소, 페이지 넘김
- **터치 제스처**: 드래그앤드롭, 스와이프

## 4. 브랜드 방향성
