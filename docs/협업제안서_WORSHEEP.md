# WORSHEEP 공식 악보 파트너십 제안서

**찬양팀을 위한 통합 악보 관리 플랫폼**

2026년 1월

---

## 인사말

안녕하세요,

먼저 하나님의 찬양 사역에 헌신하시는 모든 분들께 깊은 감사의 말씀을 드립니다.
팀에서 만드신 찬양과 악보들이 수많은 교회와 찬양팀에게 큰 은혜가 되고 있습니다.

저희는 한국 교회 찬양팀들의 악보 관리와 예배 준비를 돕기 위한 **WORSHEEP**이라는 플랫폼을 운영하고 있습니다. 이 서비스가 찬양 사역자들과 저작권자 모두에게 유익이 될 수 있도록, 귀 팀과의 협력을 제안드리고자 합니다.

## 1. 핵심 요약

### 요청드리는 것

> **귀 팀의 공식 악보를 WORSHEEP에서 사용할 수 있도록 허락해 주세요.**

### 팀이 동의하시면 얻는 것

| 혜택 | 설명 |
|------|------|
| **수익 창출** | 현재 무료로 배포 중인 악보에서 추가 수익 발생. 구독 수익의 50%를 사용량 비율로 정산 |
| **저작권 보호** | '공식 인증' 마크로 귀 팀의 악보임을 명확히 표시 |
| **통합 통계** | 어떤 곡이 인기 있는지, 어느 지역에서 많이 사용되는지 데이터 제공 |
| **홍보 효과** | 전국 교회 찬양팀에 자연스럽게 노출 |
| **무료 프리미엄** | 협력 팀에게 모든 프리미엄 기능 무료 제공 |

### 팀에서 해주실 일

> **동의서 서명뿐입니다.** 악보 등록, 관리, 업데이트는 모두 저희가 담당합니다.

## 2. 왜 지금 이 제안을 드리는가

### 2.1 현재 악보 저작권이 지켜지지 않는 현실

솔직히 말씀드리겠습니다.

**귀 팀의 악보는 이미 인터넷 곳곳에서 무단으로 유통되고 있습니다.**

- 네이버, 구글 검색으로 출처 불명의 악보들이 공유되고 있습니다
- 코드가 틀리거나 편곡된 비공식 악보들이 "공식"처럼 돌아다닙니다
- 저작권자 표기 없이 무단 복사, 재배포가 일상화되어 있습니다

이런 상황에서 **저작권을 주장하기도, 사용을 막기도 어렵습니다.**
법적 대응을 하자니 비용과 시간이 들고, 그냥 두자니 권리가 침해됩니다.

### 2.2 악보가 여기저기 흩어져 있는 문제

현재 많은 찬양팀들이 악보를 다양한 경로로 배포하고 있습니다:

- 블로그 (네이버, 티스토리)
- 인스타그램
- 페이스북 페이지
- 카카오톡 오픈채팅
- 공식 홈페이지
- 유튜브 영상 설명란

**문제점:**
- 사용자들이 공식 악보를 찾기 위해 여러 플랫폼을 돌아다녀야 합니다
- 어떤 버전이 최신인지, 공식인지 확인하기 어렵습니다
- 팀 입장에서도 여러 채널을 관리하는 것이 부담됩니다
- **얼마나 많은 사람들이 악보를 사용하는지 파악할 방법이 없습니다**

### 2.3 저작권을 지키면서 수익도 얻는 방법

**WORSHEEP의 제안은 이 문제들을 해결합니다:**

```
[현재 상황]
악보 무단 유통 → 저작권 침해 → 수익 없음 → 통계 없음

[WORSHEEP 파트너십]
공식 악보 등록 → 저작권 보호 → 수익 정산 → 상세 통계 제공
```

## 3. 파트너십의 구체적인 내용

### 3.1 저희가 하는 것

| 항목 | 내용 |
|------|------|
| **악보 등록** | 귀 팀의 공식 악보를 WORSHEEP에 등록 |
| **공식 인증** | '공식 악보' 마크를 통해 비공식 악보와 구분 |
| **저작권 표기** | 모든 악보에 저작권자 정보 자동 표기 |
| **악보 관리** | 신곡 출시 시 확인 후 등록, 업데이트 관리 |
| **수익 정산** | 유료 구독 수익의 50%를 사용량 비율로 정산 |
| **통계 제공** | 조회수, 다운로드 수, 지역별 사용 현황 등 |

### 3.2 저희가 하지 않는 것

| 항목 | 설명 |
|------|------|
| **저작권 이전** | 저작권은 100% 귀 팀에 있습니다. 저희는 **사용 허락**만 받습니다 |
| **악보 판매** | 악보 검색과 조회는 무료입니다. 악보 자체를 판매하지 않습니다 |
| **독점 계약** | 기존 배포 채널을 대체하지 않습니다. 추가 채널로 활용됩니다 |
| **수정 권한** | 악보를 임의로 수정하지 않습니다. 변경이 필요하면 팀에 요청드립니다 |

### 3.3 채보 악보에 대한 동의

귀 팀의 공식 악보가 없거나 구하기 어려운 경우, 많은 찬양팀들이 **채보자(일반 사용자)가 만든 악보**를 사용하고 있습니다.

파트너십에 동의해 주시면, 다음 사항에도 동의하시는 것으로 간주됩니다:

> **귀 팀 곡에 대해 채보자가 만든 악보를 WORSHEEP에서 제공하는 것에 동의합니다.**

#### 채보 악보 수익 분배 구조

채보자가 만든 악보가 서비스에 등록되면, 해당 곡의 사용량에 따른 수익이 다음과 같이 분배됩니다:

```
유료 구독 수익 (해당 곡 사용량 비율)
    ├─ 50% → 저작권자 (귀 팀)
    ├─ 20% → 채보자
    └─ 30% → 플랫폼 운영
```

| 수익 대상 | 비율 | 이유 |
|-----------|------|------|
| **저작권자 (귀 팀)** | 50% | 원곡의 멜로디, 코드, 가사에 대한 저작권 |
| **채보자** | 20% | 악보로 정리한 노력과 편곡에 대한 권리 |
| **플랫폼** | 30% | 서비스 운영 및 관리 비용 |

#### 왜 이 동의가 필요한가요?

- 귀 팀의 공식 악보가 있는 곡: **공식 악보만 등록** (채보 악보 불필요)
- 귀 팀의 공식 악보가 없는 곡: **채보 악보 등록 가능** (동의 필요)

채보 악보가 등록되더라도:
- 귀 팀은 **사용량의 50%를 정산**받습니다
- 악보에 **귀 팀이 원곡 저작권자임이 표기**됩니다
- 나중에 공식 악보를 제공해 주시면 **공식 악보로 대체**됩니다

#### 거부하실 수 있나요?

채보 악보 동의를 원치 않으시면 말씀해 주세요.
그 경우 귀 팀이 직접 제공해 주신 공식 악보만 등록됩니다.

단, 공식 악보가 없는 곡은 서비스에서 제공되지 않습니다.

### 3.4 수익 구조

WORSHEEP은 악보 **검색, 조회, 미리보기는 항상 무료**로 제공합니다.
부가 기능(팀 콘티 관리, 송폼 편집 등)과 무제한 다운로드는 유료 구독으로 운영됩니다.

| 무료 기능 | 유료 구독 기능 |
|-----------|----------------|
| 악보 검색 | 팀 콘티 관리 및 공유 |
| 악보 조회/미리보기 무제한 | 송폼 편집 및 저장 |
| 악보 에디터 보기 모드 | 고급 필기 도구 |
| **악보 다운로드 (주 5회 제한)** | **악보 다운로드 무제한** |
| 좋아요 / 저장 | 일괄 다운로드 |

**→ 유료 구독 수익의 50%를 저작권자들에게 사용량 비율로 정산합니다.**

예시: 월 구독 수익 1,000만원 발생 시
- 저작권자 정산 총액: 500만원
- 귀 팀 악보가 전체 사용량의 10%인 경우: 50만원 정산

### 3.5 악보 콘텐츠 구성

WORSHEEP의 악보는 세 가지로 구분됩니다:

| 분류 | 설명 | 표시 |
|------|------|------|
| **공식 악보** | 저작권자(팀)와 파트너십 체결 | '공식 인증' 마크 |
| **퍼블릭 도메인** | 저작권 만료된 곡 (저작자 사후 70년) | 자유 이용 가능 표시 |
| **사용자 업로드** | 사용자가 개인/팀용으로 등록 | 업로더 정보 표기 |

> **파트너십을 맺으시면**, 귀 팀의 악보는 '공식 악보'로 분류되어
> 비공식 악보와 명확히 구분되고, 검색 시 우선 노출됩니다.

### 3.6 사용자 업로드 악보와의 차별점

사용자들은 공식 악보로 등록되지 않은 곡을 직접 업로드할 수 있습니다:
- 개인용/팀 내 공유/전체 공개 선택 가능
- 전체 공개 시 다른 사용자들도 이용 가능

**하지만 사용자 업로드 악보는:**
- '공식 인증' 마크가 없습니다
- 저작권 확인이 되지 않은 상태입니다
- **저작권자 신고 시 3영업일 내 삭제됩니다** (저작권법 제102조)

**즉, 파트너십 없이도 귀 팀의 악보가 사용자에 의해 업로드될 수 있지만:**
- 공식으로 인정받지 못합니다
- 수익 정산을 받을 수 없습니다
- 틀린 코드, 잘못된 버전이 유통될 수 있습니다

**파트너십을 맺으시면:**
- 공식 악보로 인정받고
- 사용량에 따른 수익을 정산받으며
- 올바른 버전만 유통됩니다

| **팀 부담** | 직접 업로드/관리 필요 | 동의만 하면 저희가 관리 |

## 5. 팀의 우려에 대한 답변

### "이미 무료로 배포하고 있는데요?"

맞습니다. 하지만 그 무료 배포가 **팀에게 어떤 가치를 주고 있나요?**

- 얼마나 많은 사람들이 사용하는지 모릅니다
- 비공식 악보와 섞여서 공식 악보의 가치가 희석됩니다
- 노력에 대한 보상이 없습니다

WORSHEEP 파트너십은 **무료 배포는 유지**하면서:
- 정확한 사용 통계를 받고
- 공식 악보로서의 인정을 받고
- 추가 수익까지 얻을 수 있습니다

### "저작권이 저희한테 남나요?"

**네, 100% 남습니다.**

저희가 요청드리는 것은 **"WORSHEEP 플랫폼에서 악보를 제공할 수 있는 사용 허락"**입니다.
저작권 양도가 아닙니다.

- 다른 곳에서 악보를 배포하셔도 됩니다
- 언제든 파트너십을 종료하실 수 있습니다
- 악보의 모든 권리는 귀 팀에 있습니다

### "독점 계약인가요?"

**아닙니다.**

- 기존 블로그, SNS 배포를 계속하셔도 됩니다
- 다른 플랫폼과 계약하셔도 됩니다
- WORSHEEP은 **추가 채널**입니다

### "악보 등록은 저희가 해야 하나요?"

**아닙니다.**

귀 팀에서 해주실 일은 **동의서 서명뿐**입니다.

- 악보 등록: 저희가 합니다
- 정보 입력: 저희가 합니다
- 업데이트 관리: 저희가 합니다
- 신곡 등록: 공식 채널 확인 후 저희가 합니다

필요시 파일만 전달해 주시면 됩니다.

### "베타에 이미 저희 악보가 있던데요?"

현재 베타 테스트 기간에는 서비스 검증을 위해 인기 CCM 악보들을 임시로 등록해 두었습니다.

**하지만 정식 서비스 출시 시:**
- 파트너십이 체결되지 않은 곡은 **모두 삭제**됩니다
- **공식 파트너 악보**와 **퍼블릭 도메인 악보**만 남습니다

즉, 파트너십 없이는 정식 서비스에서 귀 팀의 악보가 제공되지 않습니다.
지금 베타 단계에서 파트너십을 맺으시면:
- 정식 출시 시 바로 공식 악보로 서비스됩니다
- 베타 기간 동안의 사용 데이터도 제공받으실 수 있습니다

## 7. WORSHEEP 서비스 소개

### 7.1 서비스 개요

WORSHEEP은 한국 교회 찬양팀을 위한 종합 악보 관리 플랫폼입니다.

- **현재 등록 곡 수**: 1,000곡 이상
- **베타 테스트**: 2026년 상반기 (약 20개 팀, 50명 사용자)
- **정식 서비스**: 2026년 하반기 예정

### 7.2 주요 기능

| 기능 | 설명 |
|------|------|
| **통합 악보 검색** | 곡명, 아티스트, 가사, Key, 박자, 템포, 테마별 검색 |
| **팀 콘티 관리** | 예배 순서 저장, 팀원 공유, 히스토리 관리 |
| **송폼 편집** | Verse, Chorus, Bridge 등 구간 표시 및 편집 |
