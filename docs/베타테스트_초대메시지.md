## 1. 카카오톡용 (짧은 버전)

```

제가 찬양팀을 위한 악보 관리 서비스를 만들고 있는데, 베타 테스트에 참여해주실 분을 찾고 있어요.

✅ 악보 검색/다운로드
✅ 콘티 만들기
✅ 악보에 필기 (송폼, 파트 표시)
✅ 팀원과 공유

이런 기능들이 있고, 베타 기간에는 전부 무료예요!

혹시 관심 있으시면 사용해보시고 피드백 주시면 감사하겠습니다 🙏

👉 https://worsheep.org

*저작권 문제로 링크는 다른 분들께 공유하지 말아주세요!
```

```
베타 기간 동안 모든 기능 무료입니다!
팀원분들과 함께 써보시고 피드백 주시면 큰 도움이 됩니다.
```

## 2. 이메일용 (상세 버전)

### 제목 예시
- WORSHEEP 베타 테스트 초대
- [초대] 찬양팀 악보 관리 서비스 베타 테스트
- 찬양팀을 위한 새로운 서비스, 베타 테스트에 초대합니다

### 본문

```
안녕하세요, [이름]님!

먼저 찬양 사역에 헌신하시는 모습에 감사드립니다.

저는 교회 찬양팀을 위한 악보 관리 서비스 'WORSHEEP'을 개발하고 있는 조민서입니다.
이번에 베타 테스트를 시작하게 되어, [이름]님을 초대드리고자 연락드렸습니다.


📌 WORSHEEP이란?

찬양팀의 예배 준비를 돕는 올인원 악보 관리 플랫폼입니다.

• 악보 검색: 1,700곡+ CCM 악보를 한 곳에서 검색
• 콘티 제작: 드래그 앤 드롭으로 예배 순서 편집
• 악보 필기: 송폼(V, C, B), 파트 태그, 메모 추가
• 팀 협업: 팀원과 콘티/악보 실시간 공유
• 다운로드: PDF, PPT로 내보내기


🎁 베타 테스터 혜택

• 베타 기간(~6월) 동안 모든 기능 무료
• 정식 출시 후 Pro 구독 3개월 50% 할인
• 베타 테스터 전용 뱃지
• 피드백 우선 반영


✅ 참여 방법

1. 아래 링크에서 회원가입
2. 서비스 자유롭게 사용
3. 불편한 점이나 개선 아이디어가 있으면 피드백

👉 https://worsheep.org


⚠️ 부탁드리는 점

현재 베타 버전은 저작권 허락 절차가 진행 중이라,
서비스 링크를 외부에 공유하지 말아 주시기 바랍니다.
추가로 참여하고 싶은 분이 계시면 저에게 직접 연락 부탁드립니다.

• 서비스 내 피드백 버튼 (우측 하단)
• 이메일: minseo@worsheep.org
```
