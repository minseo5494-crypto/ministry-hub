### 1.3 베타 테스트 주의사항

> **중요: 서비스 링크 공유를 삼가해 주세요.**

베타 기간 동안 서비스 링크를 외부에 공유하지 말아 주시기 바랍니다.

**이유:**
- 현재 등록된 악보들은 **저작권 허락 절차가 진행 중**입니다
- 정식 출시 전까지 **비공개 테스트** 형태로 운영됩니다
- 외부 유출 시 저작권 문제가 발생할 수 있습니다

**가능한 것:**
- 팀원 초대 (초대 코드 공유)
- 팀 내 콘티 공유

**삼가해 주실 것:**
- SNS, 블로그 등에 서비스 링크 게시
- 불특정 다수에게 서비스 소개

베타 테스트에 추가로 참여하고 싶은 분이 계시다면, 담당자에게 연락해 주세요.
정식 출시 후에는 자유롭게 공유하실 수 있습니다. 양해 부탁드립니다.

## 2. 시작하기

### 2.1 회원가입

1. https://worsheep.com 접속
2. "회원가입" 클릭
3. 이메일 또는 Google 계정으로 가입
4. 이용약관 및 저작권 정책 동의
5. 이메일 인증 완료

### 2.2 홈 화면에 앱 추가 (PWA)

WORSHEEP을 앱처럼 사용할 수 있습니다. 홈 화면에 추가하면 브라우저 없이 바로 실행됩니다.

#### iPhone / iPad (Safari)
1. Safari에서 https://worsheep.com 접속
2. 하단 **공유 버튼** (사각형 + 화살표) 탭
3. **"홈 화면에 추가"** 선택
4. 이름 확인 후 **"추가"** 탭

#### Android (Chrome)
1. Chrome에서 https://worsheep.com 접속
2. 우측 상단 **점 3개 메뉴** 탭
3. **"홈 화면에 추가"** 또는 **"앱 설치"** 선택
4. **"설치"** 또는 **"추가"** 탭

#### PC (Chrome, Edge)
1. 브라우저에서 https://worsheep.com 접속
2. 주소창 오른쪽 **설치 아이콘** 클릭 (모니터 + 화살표 모양)
3. **"설치"** 클릭

> **장점**: 앱처럼 전체 화면으로 사용 가능하고, 홈 화면에서 바로 실행할 수 있습니다.

### 2.3 팀 생성/참여

#### 새 팀 만들기
1. 로그인 후 "팀 관리" 메뉴
2. "새 팀 만들기" 클릭
3. 팀 이름 입력
4. 초대 코드 생성 → 팀원에게 공유

#### 기존 팀 참여하기
1. 팀 리더에게 초대 코드 받기
2. "팀 관리" → "팀 참여하기"
3. 초대 코드 입력

## 3. 주요 기능 사용법

### 3.1 악보 검색

**기본 검색**
- 곡 제목, 아티스트명, 가사로 검색
- 검색창에 입력 후 Enter 또는 검색 버튼

**필터 검색**
- Key: 원하는 조성 선택
- 박자: 4/4, 3/4, 6/8 등
- 템포: 느린, 보통, 빠른
- 테마: 감사, 찬양, 회개, 치유 등
- 시즌: 사순절, 부활절, 성탄절 등

**AI 검색**
- 자연어로 검색 가능
- 예: "부활절에 부르기 좋은 밝은 찬양"
- 예: "4/4박자 D키 찬양 추천해줘"

### 3.2 악보 보기

1. 검색 결과에서 곡 클릭
2. 악보 뷰어에서 확인
3. 확대/축소: 핀치 또는 버튼
4. 페이지 넘기기: 좌우 스와이프 또는 화살표

### 3.3 콘티 만들기

1. "콘티" 메뉴 클릭
2. "새 콘티 만들기"
3. 콘티 정보 입력 (날짜, 예배 유형)
4. 악보 검색 → 곡 추가
5. 드래그로 순서 변경
6. 저장

**콘티 공유**
- 팀원과 자동 공유 (같은 팀)
- 링크 공유도 가능

### 3.4 악보 필기 (에디터)

1. 악보 뷰어에서 "편집" 버튼
2. 도구 선택:
   - **펜**: 자유 필기
   - **형광펜**: 강조 표시
   - **텍스트**: 메모 추가
   - **송폼**: V, C, B 등 구간 표시
   - **파트 태그**: Vocal, Bass 등 파트 표시
3. 저장 → 자동 동기화

### 3.5 다운로드

**개별 다운로드**
- 악보 뷰어에서 다운로드 버튼
- PDF, 이미지 선택

**콘티 일괄 다운로드**
- 콘티 페이지에서 "전체 다운로드"
- PDF 또는 PPT 선택
