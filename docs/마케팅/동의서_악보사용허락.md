# 악보 사용 허락 동의서

## 팀(저작권자)용

---

### 동의서 양식 (이메일 회신용)

아래 내용을 이메일로 회신해 주시면 동의 완료됩니다.

```
제목: [WORSHEEP] 악보 사용 허락 동의 - [팀명]

안녕하세요, [팀명] 입니다.

WORSHEEP 악보 사용 허락에 동의합니다.

1. 동의 내용
   - 당 팀의 공식 악보를 WORSHEEP 플랫폼에서 제공하는 것에 동의합니다.
   - 공식 악보가 없는 곡에 대해 채보자가 만든 악보를 제공하는 것에 동의합니다.
   - 수익 분배 비율(저작권자 50%, 채보자 20%, 플랫폼 30%)에 동의합니다.

2. 팀 정보
   - 팀명:
   - 담당자명:
   - 연락처(이메일):
   - 연락처(전화, 선택):

3. 정산 정보 (정식 서비스 시 필요, 현재는 선택)
   - 예금주:
   - 은행명:
   - 계좌번호:

동의일: 2026년   월   일

[팀명/담당자명]
```

### 동의서 양식 (문서 서명용)

```
                    악보 사용 허락 동의서

본인(팀)은 WORSHEEP(이하 "서비스")에서 아래 내용에 동의합니다.


제1조 (동의 내용)

1. 본 팀이 저작권을 보유한 악보를 서비스에서 제공하는 것에 동의합니다.
2. 본 팀 곡에 대해 채보자가 제작한 악보를 서비스에서 제공하는 것에 동의합니다.
3. 서비스 이용자가 악보를 검색, 조회, 다운로드할 수 있도록 허락합니다.


제2조 (저작권)

1. 본 동의는 저작권 양도가 아닌 사용 허락입니다.
2. 악보의 모든 저작권은 본 팀에게 있습니다.
3. 본 팀은 기존 배포 채널을 계속 운영할 수 있습니다.


제3조 (수익 분배)

유료 구독 수익은 아래 비율로 분배됩니다:
- 저작권자(본 팀): 50%
- 채보자 (해당 시): 20%
- 플랫폼 운영: 30%

* 공식 악보의 경우 채보자 몫은 저작권자에게 귀속됩니다.


제4조 (동의 철회)

1. 본 팀은 언제든지 서면으로 동의를 철회할 수 있습니다.
2. 철회 시 14일 이내에 해당 악보가 서비스에서 삭제됩니다.


제5조 (기타)

1. 본 동의서에 명시되지 않은 사항은 상호 협의하여 결정합니다.
2. 정식 서비스 출시 시 정식 계약서로 대체될 수 있습니다.


                    동의 정보

팀명:
담당자:
연락처:
동의일:         년      월      일

                    서명:


                    서비스 정보

서비스명: WORSHEEP
담당자: 조민서
연락처: support@worsheep.org / 010-3150-4221
```

## 사용 가이드

### 이메일 동의 받는 경우

1. 협업 제안서를 먼저 발송
2. 관심 표명 시 동의서 양식 안내
3. 이메일 회신으로 동의 수집
4. 동의 이메일 보관 (법적 증거)

### 문서 서명 받는 경우

1. 협업 제안서와 동의서 함께 발송
2. 서명된 동의서 수령 (스캔/사진 가능)
3. 원본 보관

### 동의 관리

- 모든 동의 내역은 스프레드시트로 관리
- 동의일, 동의자, 연락처, 악보 목록 기록
- 정식 서비스 시 정식 계약서로 전환
