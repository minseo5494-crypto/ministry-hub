# 파트너 컨택 메시지 템플릿

> 찬양 사역팀에게 보내는 파트너십 제안 메시지입니다.
> 목표: **미팅 약속 잡기** (설득은 만나서)

---

## 메인 컨택 메시지 (이메일)

```
제목: [팀명] 악보가 예배 현장에서 쓰일 때마다, 팀에게 수익이 돌아간다면

안녕하세요, [팀명] 팀 관계자님.

악보 관리 플랫폼 WORSHEEP을 운영하고 있는 조민서입니다.

저는 교회에서 10년 넘게 찬양팀으로 섬기면서
한 가지 이상한 점을 느꼈습니다.

음원은 스트리밍될 때마다 수익이 발생하는데,
악보는 아무리 많이 쓰여도 만든 분에게 돌아가는 게 없습니다.
제도 자체가 없어요.

CCM은 일반 음악 산업과 다릅니다.
악보의 저작권이 음반사나 유통사가 아니라 팀에게 있고,
한국에서 CCM을 이끄는 팀이 몇 팀 안 됩니다.
그런데도 어노인팅을 제외하면 악보로 수익을 만들고 있는 팀이
거의 없는 게 현실입니다.

대기업이나 기존 음악 플랫폼은 기독교 시장에 들어오기 어렵습니다.
시장 규모가 작아 보이니까요.
하지만 저는 이 안에서 예배하는 사람으로서,
이 문제를 풀 수 있는 건 이 안에 있는 사람이라고 생각합니다.

WORSHEEP은 찬양팀이 악보를 검색하고 콘티를 만드는 플랫폼입니다.
팀원들이 매주 예배를 준비하면서 자연스럽게 데이터가 쌓이고,
그 데이터를 기반으로 곡을 만든 사역팀에게 수익이 정산됩니다.

음반사를 거치지 않고 팀과 직접 계약하기 때문에
수익이 중간에서 빠지지 않고 팀에게 돌아갑니다.

팀에서 특별히 하실 일은 없습니다.
공식 악보의 사용 허가만 해주시면,
나머지는 저희가 다 합니다.
저작권 양도가 아니고, 독점도 아니고, 기존 활동에 영향을 주는 것도 없습니다.
그냥 수익 파이프라인이 하나 생기는 겁니다.

30분만 시간 내주시면,
제안서와 함께 구체적인 구조를 설명드리겠습니다.
오프라인이든 온라인이든, 편하신 방식으로 맞추겠습니다.

편하신 시간대를 알려주시면 찾아뵙겠습니다.

감사합니다.

조민서 드림
WORSHEEP 대표
minseo@worsheep.org | 010-3150-4221
worsheep.org

[첨부: 베타 파트너 프로그램 제안서.pdf]
```

## 채널별 축약 버전

### 카톡/DM (짧은 버전)

> 위 이메일의 핵심만 추려서 보냅니다.
> 관심을 얻으면 이메일 주소를 받아 정식 제안을 보냅니다.

```
안녕하세요, [팀명] 팀 관계자님.
악보 관리 플랫폼 WORSHEEP의 조민서입니다.

음원은 스트리밍되면 수익이 생기는데,
악보는 아무리 쓰여도 만든 분에게 돌아가는 게 없잖아요.

저희는 찬양팀들이 악보를 관리하는 플랫폼을 만들고 있고,
거기서 쌓이는 사용 데이터를 기반으로
곡을 만든 사역팀에게 수익을 정산하는 구조예요.

팀에서 하실 일은 없고, 악보 사용 허가만 해주시면 됩니다.
저작권 양도도 독점도 아니에요.

30분 정도 설명드릴 기회를 주시면 감사하겠습니다.
이메일 주소 알려주시면 제안서 보내드릴게요!
```

### 인스타그램 DM (초간단)

```
안녕하세요! 악보 관리 플랫폼 WORSHEEP입니다.

[팀명] 팀 곡들이 예배 현장에서 정말 많이 불리고 있는데,
그 사용 데이터가 팀에게 전혀 돌아가지 않는 게 아쉬워서 연락드려요.

저희가 그걸 데이터로 모아서 수익으로 정산해 드리는 프로그램을 운영하고 있거든요.
저작권 양도 같은 건 전혀 없습니다.

담당자분 이메일을 알 수 있을까요?
상세 제안서 보내드리겠습니다!
```

## 팔로업 메시지

### 첫 연락 후 3일 무응답 시

```
안녕하세요, 지난번 연락드렸던 WORSHEEP 조민서입니다.
혹시 제안서 확인하셨을까요?

궁금하신 점이 있으시면 편하게 말씀해 주세요.
간단한 통화(10분)로도 설명드릴 수 있습니다!
```

### 미팅 확정 후

```
미팅 잡아주셔서 감사합니다!

- 일시: [날짜] [시간]
- 장소: [장소] / 온라인: [Zoom/Google Meet 링크]
- 소요: 약 30분
- 준비물: 없습니다 (제안서는 제가 준비해 갑니다)

당일 뵙겠습니다!
```

## 사용 가이드

**핵심: 미팅을 잡는 게 목표입니다.**

| 상황 | 사용 템플릿 |
|------|-----------|
| 이메일 주소를 아는 경우 | 메인 컨택 메시지 (이메일) |
| 카톡/DM만 가능한 경우 | 채널별 축약 버전 → 이메일 받으면 메인 발송 |
| 인스타/유튜브 DM | 인스타그램 DM → 이메일 받으면 메인 발송 |
| 무응답 | 팔로업 메시지 |

**원칙:**
- "왜 악보에는 수익 구조가 없는가"로 시작
- "팀에서 할 일은 없다"를 반드시 전달
- 첫 연락에 제안서를 바로 보내지 말 것 (이메일 제외)
- 만나서 설명하면 된다 — 메시지의 목표는 **미팅 약속**
